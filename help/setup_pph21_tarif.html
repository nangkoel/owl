SETUP PPH 21 TARIF

Modul : SDM
Tentang : Setup PPh 21 Tarif
Isi :

Langkah-langkah Setup PPh 21 Tarif :

Masuk ke menu, akan terdapat 3 tab yaitu Tab PTKP/Tahun Form, Tab Kontribusi/Tahun dan Tab Biaya Jabatan seperti di bawah ini :

  1. Tab PTKP/Tahun Form
    1. Klik Tab PTKP/Tahun Form, isikan Penghasilan Tidak Kena Pajak (PTKP) single, tanpa anak, anak satu, anak dua dan anak tiga kemudian klik tombol Simpan.

    2. Setelah klik tombol Simpan akan muncul pesan berikut :

      3. Klik tombol OK.

  2. Tab Kontribusi/Tahun
    1. Klik Tab Kontribusi/Tahun, isikan level, rupiah dan tarif (%) kemudian klik tombol Simpan.

    2. Setelah klik tombol Simpan akan muncul pesan berikut :

      3. Klik tombol OK.

  3. Tab Biaya Jabatan
    1. Klik Tab Biaya Jabatan, isikan porsi (%) dan Maximum rupiah/bulan kemudian klik tombol Simpan.

    2. Akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK, maka akan muncul pesan berikut :

      Klik tombol OK.
