TRANSAKSI KELUAR MASUK BIBIT

Modul : Kebun
Tentang : Transaksi Keluar Masuk Bibit
Isi :

Langkah-langkah Input Transaksi Keluar Masuk Bibit :

Masuk ke menu, akan muncul 5 tab menu seperti di bawah ini :

  1. Tab Penyemaian
    1. Klik Tab Penyemaian

      2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk menyimpan atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan :

      3. Kode Transaksi TMB = Tanam Bibit. Terdapat fasilitas pencarian berdasarkan tanggal, batch dan status.

    3. Klik icon untuk edit data penyemaian bibit.
    4. Klik icon untuk hapus data penyemaian bibit.
    5. Klik icon untuk posting data penyemaian bibit.
  2. Tab Doubletoon
    1. Klik Tab Doubletoon

      2. Double toon merupakan kecambah yang memiliki titik tumbuh lebih dari satu, seperti 2,3 atau 4 titik tumbuh dalam satu kecambah.

      Batch yang dipilih pada doubletoon harus sudah diposting terlebih dahulu pada tab penyemaian.

      Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data doubletoon atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan :

      3. Kode Transaksi DBT = Doubletoon. Terdapat fasilitas pencarian berdasarkan tanggal, batch dan status.

    3. Klik icon untuk edit data doubletoon.
    4. Klik icon untuk hapus data doubletoon.
    5. Klik icon untuk posting data doubletoon.
  3. Tab Afkir Bibit
    1. Klik Tab Afkir Bibit

      2. Afkir Bibit merupakan bibit yang tidak sempurna.

      Batch yang dipilih pada afkir bibit harus sudah diposting terlebih dahulu pada tab penyemaian.

      Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data afkir bibit atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan :

      3. Kode Transaksi AFB = Afkir Bibit. Terdapat fasilitas pencarian berdasarkan tanggal, batch dan status.

    3. Klik icon untuk edit data afkir bibit.
    4. Klik icon untuk hapus data afkir bibit.
    5. Klik icon untuk posting data afkir bibit.
  4. Tab Transplanting/ Mutasi Bibit
    1. Klik Tab Transplanting Bibit

      2. Transplanting Bibit merupakan menanam di Polybag Besar.

      Batch yang dipilih pada transplanting bibit harus sudah diposting terlebih dahulu pada tab penyemaian.

      Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data afkir bibit atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan :

      3. Kode Transaksi TPB = Transplanting. Terdapat fasilitas pencarian berdasarkan tanggal, batch dan status.

    3. Klik icon untuk edit data transplanting bibit.
    4. Klik icon untuk hapus data transplanting bibit.
    5. Klik icon untuk posting data transplanting bibit.
  5. Tab Surat Pengiriman Bibit
    1. Klik Tab Pengiriman Bibit

      2. Batch yang dipilih pada transplanting bibit harus sudah diposting terlebih dahulu pada tab penyemaian.

      Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data pengiriman bibit atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan :

      3. Kode Transaksi PNB = Pengiriman Bibit

    3. Klik icon untuk hapus data pengiriman bibit.
    4. Klik icon untuk posting data pengiriman bibit. Setelah melakukan posting, data tidak dapat dihapus.
    5. Klik icon untuk melihat data pengiriman bibit dalam format PDF.
  6. Tab Stok Detail
