Langkah-langkah input Transaksi Produksi PKS :

Masuk ke menu, akan terdapat Form Entry Data dan Form Tutup seperti di bawah ini :

  1. Form Entry Data
    1. Isikan tahun budget, pilih unit dan status buah (External, Internal, Afiliasi) kemudian klik tombol Simpan.

    2. Akan muncul form sebaran bulanan, lengkapi data sebaran bulanan seperti di bawah ini :

    3. Klik icon untuk simpan sebaran bulanan produksi PKS.
    4. Data sebaran bulanan yang sudah tersimpan akan muncul pada List Data di bawah ini :

    5. Klik pada salah satu baris data untuk edit data sebaran bulanan.
    6. Klik icon untuk hapus data produksi PKS.
  2. Form Tutup
    1. Setelah selesai mengisi budget maka harus dilakukan finalisasi untuk budget tersebut agar tidak dapat dirubah lagi dan tidak mempengaruhi data yang lain.
    2. Pilih tahun budget dan unit kemudian klik tombol Tutup.

    3. Akan muncul konfirmasi di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses.
