Langkah-langkah Setup Hari Kerja Efektif :

  1. Masuk ke menu, pilih periode, isikan jumlah hari minggu, jumlah hari libur dan catatan kemudian klik tombol Simpan. Untuk HK Efektif akan otomatis dihitung oleh sistem.

  2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  3. Klik icon untuk hapus data hari kerja efektif.
