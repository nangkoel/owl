LAPORAN TOTAL PEMBELIAN BARANG

Modul : Gudang Laporan
Tentang : Laporan Total Pembelian Barang
Isi :

Langkah-langkah melihat Laporan Total Pembelian Barang :

  1. Masuk ke menu, pilih periode dan isikan kurs (optional) kemudian klik tombol Preview untuk menampilkan laporan.

    2. Laporan dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

  2. Berikut tampilan dari Laporan Total Pembelian Barang :

  3. Klik salah satu baris untuk melihat detail per kelompok pembelian barang. Berikut tampilan detail dari pembelian barang kapital :

    4. Klik tombol Back untuk kembali ke halaman sebelumnya atau klik tombol Excel untuk melihat detail dalam format Excel.

  4. Berikut tampilan detail dari pembelian barang non kapital :

    5. Klik tombol Back untuk kembali ke halaman sebelumnya atau klik tombol Excel untuk melihat detail dalam format Excel.

    Untuk pembelian barang non kapital dapat dilihat detail per kelompok barang nya dengan klik pada salah satu baris di atas. Berikut tampilan nya :

    Klik tombol Back untuk kembali ke halaman sebelumnya atau klik tombol Excel untuk melihat detail dalam format Excel.
