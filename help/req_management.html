REQUEST MANAGEMENT

Modul : IT
Tentang : Request Management
Isi :

Langkah-langkah Request Management :

  1. Masuk ke menu, akan muncul semua list permintaan layanan :

  2. Klik icon untuk melihat detail permintaan layanan.
  3. Pilih pelaksana (karyawan IT) dan isikan standard jam kemudian klik tombol Simpan. Secara otomatis sistem akan mengirimkan email pemberitahuan kepada pelaksana.

    4. Berikut isi email tersebut :

    Karyawan a/n: [nama karyawan] meminta layanan [jenis layanan] pada tanggal [tanggal] ke departemen IT
    dengan deskripsi [deskripsi]
    Mohon dibantu, dan jika sudah selesai jangan lupa update status pelaksanaannya dari menu IT->request response

    Jika ditolak maka klik tombol tolak dan isikan alasan penolakannya, berikut tampilan form komentar penolakan :
