TRANSAKSI KEGIATAN PANEN

Modul : Kebun
Tentang : Transaksi Kegiatan Panen
Isi :

Langkah-langkah Transaksi Kegiatan Panen :

  1. Masuk ke menu, klik icon untuk membuat transaksi pembukaan lahan.
  2. Berikut tampilan form tambah header :

    3. Lengkapi data seperti di atas kemudian klik tombol Simpan. No. Transaksi akan terbentuk secara otomatis seperti di bawah ini :

  3. Setelah simpan data header maka akan muncul form detail seperti di bawah ini :

    4. Lengkapi data seperti di atas kemudian klik tombol Simpan. Setelah simpan detail prestasi maka data tersebut akan muncul pada tabel prestasi di bawah ini :

  4. Klik icon untuk edit data transaksi kegiatan panen.
  5. Klik icon untuk menghapus data transaksi kegiatan panen.
  6. Klik icon untuk meihat semua list data transaksi pemeliharaan TM atau lakukan pencarian data seperti di bawah ini :

    7. Klik tombol Cari untuk melakukan pencarian.

  7. Berikut hasil dari pencarian di atas :

  8. Klik icon pada form cetak untuk mencetak list data transaksi kegiatan panen.
  9. Klik icon pada form cetak untuk melihat list transaksi kegiatan panen dalam format PDF.
  10. Klik icon untuk edit data transaksi kegiatan panen.
  11. Klik icon untuk menghapus data transaksi kegiatan panen.
  12. Klik icon untuk posting transaksi kegiatan panen. Setelah dilakukan posting, icon akan berubah menjadi berwarna hijau
  13. Klik icon untuk melihat data transaksi kegiatan panen dalam format PDF.
