LAPORAN MUTASI STOCK

Modul : Gudang Laporan
Tentang : Laporan Mutasi Stock
Isi :

Langkah-langkah melihat Laporan Mutasi Stock :

  1. Masuk ke menu, pilih gudang, kelompok barang, periode, per mayor, urut berdasarkan dan urut Asc/Desc kemudian klik tombol Proses untuk menampilkan laporan.

    2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

  2. Berikut tampilan dari Laporan Mutasi Stock Per Mayor (Per kelompok barang):

    3. Jika Per Mayor nya tidak dipilih, maka akan menampilkan laporan per kode barang :

  3. Untuk melihat detail transaksi penerimaan, klik pada baris Masuk untuk periode tertentu :

    4. Klik icon untuk melihat detail transaksi penerimaan dalam format Excel.

  4. Untuk melihat detail transaksi pengeluaran, klik pada baris Keluar untuk periode tertentu :

    5. Klik icon untuk melihat detail transaksi pengeluaran dalam format Excel.
