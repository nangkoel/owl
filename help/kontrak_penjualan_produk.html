KONTRAK PENJUALAN

Kontrak sudah diinput di timbangan tetapi belum diinput di Jakarta.

Input Kontrak Penjualan oleh Dept. Pemasaran Jakarta:

  1. Input kontrak di program timbangan (bisa diwakilkan oleh Kasie PKS)
  2. Input kontrak di OWL (oleh pemasaran Jakarta)

    3. Masuk ke menu, maka akan terdapat 2 tab yaitu tab Form dan tab List Data seperti di bawah ini :

    1. Tab Form
      1. Klik pada Tab Form maka akan terdapat Form Kop, Form Informasi Kustomer, Form Order Information dan Form Catatan.
      2. Pada Form Kop, pilih nama organisasi, isikan no.kontrak dan tanggal kontrak seperti di bawah ini :

      3. Pada Form Informasi Kustomer, pilih nama pelanggan maka akan muncul informasi untuk contact person dan no. fax seperti di bawah ini :

      4. Pada Form Order Information, lengkapi data seperti di bawah ini :

      5. Pada Form Catatan, isikan catatan jika terdapat catatan.

        6. Klik tombol Simpan untuk menyimpan data kontrak penjualan.
        Klik tombol Copy Data Terakhir untuk copy data kontrak penjualan yang dibuat terakhir.
        Klik tombol Buat Baru untuk membuat kontrak penjualan baru.

    2. Tab List Data
      1. Klik tab List Data maka akan muncul semua list data kontrak penjualan.
      2. Terdapat fasilitas pencarian kontrak penjualan seperti di bawah ini :

      3. Berikut hasil pencariannya :

      4. Klik icon untuk edit data kontrak penjualan.
      5. Klik icon untuk hapus data kontrak penjualan.
      6. Klik icon untuk melihat kontrak penjualan dalam format PDF.
