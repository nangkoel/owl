LAPORAN PENJUALAN

Modul : Pemasaran
Tentang : Laporan Penjualan
Isi :

Langkah-langkah melihat Laporan Penjualan :

Masuk ke menu, akan terdapat 2 tab yaitu tab Laporan Penjualan Bulanan dan tab Laporan Penjualan Harian seperti di bawah ini :

  1. Tab Laporan Penjualan Bulanan
    1. Klik Tab Laporan Penjualan Bulanan.

      2. Pilih periode, nama perusahaan dan komoditi kemudian klik tombol Preview untuk menampilkan Laporan Penjualan Bulanan.

      Laporan ini juga dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel.

    2. Berikut tampilan dari Laporan Penjualan Bulanan :

    3. Klik salah satu baris untuk melihat Detailnya :

      4. Detail ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel.

  2. Tab Laporan Penjualan Harian
    1. Klik Tab Laporan Penjualan Harian.

      2. Pilih Pabrik, Komoditi, dan tanggal kemudian klik tombol Preview untuk menampilkan Laporan Penjualan Harian.

      Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel.

    2. Berikut tampilan dari Laporan Penjualan Harian :
