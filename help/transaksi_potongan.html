TRANSAKSI POTONGAN

Modul : SDM
Tentang : Transaksi Potongan
Isi :

Langkah-langkah input Transaksi Potongan :

  1. Masuk ke menu, klik icon untuk input transaksi potongan baru, berikut tampilan form header potongan :

    2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk menyimpan data header.

  2. Setelah header disimpan maka akan muncul form detail seperti di bawah ini :

  3. Data detail yang sudah tersimpan akan muncul pada Tabel Detail di bawah ini :

  4. Klik icon untuk edit data detail.
  5. Klik icon untuk hapus data detail.
  6. Klik icon untuk melihat semua list data potongan.
  7. Terdapat fasilitas pencarian berdasarkan keterangan seperti di bawah ini :

    8. Klik tombol Cari untuk melakukan pencarian.

  8. Berikut hasil pencariannya :

  9. Klik icon untuk mencetak list data potongan.
  10. Klik icon untuk membuka list data potongan dalam format PDF.
  11. Klik icon untuk edit keterangan header potongan.
  12. Klik icon untuk hapus data potongan.
