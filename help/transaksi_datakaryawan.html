TRANSAKSI DATA KARYAWAN

Modul : SDM
Tentang : Transaksi Data Karyawan
Isi :

Langkah-langkah input Transaksi Data Karyawan :

Masuk ke menu, klik icon untuk input karyawan baru. Akan terdapat form input dengan 7 tab yaitu tab karyawan baru, tab pengalaman kerja, tab pendidikan, tab kursus, tab keluarga, tab photo dan tab alamat seperti di bawah ini :

  1. Tab Karyawan Baru
    1. Klik Tab Karyawan Baru, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data karyawan.

    2. Saat melakukan proses simpan, akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK maka akan muncul pesan seperti di bawah ini :

      Klik tombol OK maka tab karyawan baru akan berubah menjadi nama karyawan baru tersebut :

  2. Tab Pengalaman Kerja
    1. Klik Tab Pengalaman Kerja, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data pengalaman kerja.

    2. Data yang sudah disimpan akan muncul pada list di bawah ini :

    3. Klik icon untuk hapus data pengalaman kerja.
  3. Tab Pendidikan
    1. Klik Tab Pendidikan, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data pendidikan.

    2. Data yang sudah disimpan akan muncul pada list di bawah ini :

    3. Klik icon untuk hapus data pendidikan.
  4. Tab Kursus
    1. Klik Tab Kursus, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data kursus.

    2. Data yang sudah disimpan akan muncul pada list di bawah ini :

    3. Klik icon untuk hapus data kursus.
  5. Tab Keluarga
    1. Klik Tab Keluarga, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data keluarga.

    2. Data yang sudah disimpan akan muncul pada list di bawah ini :

    3. Klik icon untuk edit data keluarga.
    4. Klik icon untuk hapus data keluarga.
  6. Tab Photo
    1. Klik Tab Photo, klik tombol Browse untuk mencari lokasi foto yang akan diupload. Ukuran foto maksimal 50Kb.

      2. Klik tombol Simpan untuk menyimpan foto atau klik tombol Batal untuk membatalkan.

    2. Setelah disimpan, foto akan tampil pada Tab Photo :

  7. Tab Alamat
    1. Klik Tab Alamat. Jika karyawan masih memiliki alamat yang lain, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data alamat.

    2. Saat menyimpan data, akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses.

    3. Data yang sudah tersimpan akan muncul pada list di bawah ini :

    4. Klik icon untuk hapus data alamat.
  8. Klik icon untuk melihat list data karyawan yang ada.
    1. Terdapat fasilitas pencarian berdasarkan nama karyawan, lokasi tugas, tipe karyawan, dan status seperti di bawah ini :

    2. Berikut hasil dari pencarian :

    3. Klik icon untuk edit data karyawan.
    4. Klik icon untuk hapus data karyawan.
    5. Klik icon untuk melihat detail data karyawan :

    6. Klik icon untuk melihat data karyawan dalam format PDF.
