SETUP MATA UANG DAN KURS

Modul : Setup
Tentang : Setup Mata Uang dan Kurs
Isi :

Langkah-langkah Setup Mata Uang dan Kurs :

  1. Masuk ke menu, isikan kode jurnal, mata uang, simbol dan kode ISO kemudian klik icon untuk menyimpan header mata uang.

  2. Klik icon untuk edit data header mata uang.
  3. Klik icon untuk hapus data header mata uang.
  4. Klik icon untuk menampilkan detail mata uang.

  5. Untuk input detail mata uang, isikan tanggal, jam dan kurs kemudian klik icon untuk menyimpan detail mata uang.

  6. Klik icon untuk edit data detail mata uang.
  7. Klik icon untuk hapus data detail mata uang.
