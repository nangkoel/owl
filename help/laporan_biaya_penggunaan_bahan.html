LAPORAN BIAYA PENGGUNAAN BAHAN

Modul : Traksi
Tentang : Laporan Biaya Penggunaan Bahan
Isi :

Langkah-langkah melihat Laporan Biaya Penggunaan Bahan :

  1. Masuk ke menu, pilih kode traksi dan periode kemudian klik tombol Preview untuk menampilkan laporan.

    2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel.

  2. Berikut tampilan dari Laporan Biaya Penggunaan Bahan :

  3. Klik salah satu baris pada kolom Jlh Keluar Gudang untuk melihat Detail Jumlah Keluar dari Gudang :

  4. Klik salah satu baris pada kolom Jlh.Pakai untuk melihat Detail Pemakaian :
