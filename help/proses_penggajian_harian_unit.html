PROSES PENGGAJIAN HARIAN UNIT

Modul : SDM
Tentang : Proses Penggajian Harian Unit
Isi :

Langkah-langkah melakukan Proses Penggajian Harian Unit:

  1. Masuk ke menu, pilih periode gaji kemudian klik tombol List Data untuk memunculkan list data gaji karyawan harian unit.

  2. Berikut tampilan list data gaji karyawan harian unit :

    3. Klik tombol Proses untuk melakukan posting.

  3. Akan muncul pesan seperti di bawah ini :

    4. Klik tombol OK.
