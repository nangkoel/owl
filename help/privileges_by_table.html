Tentang : Pengaturan Hak Akses berdasarkan Menu
Isi :

Langkah-langkah Pengaturan Hak Akses berdasarkan Menu :

  1. Masuk ke menu, akan muncul form seperti di bawah ini :

  2. Misalkan akan memberikan hak akses menu HELP untuk beberapa user maka klik menu HELP pada form di atas, akan muncul list data user seperti di bawah ini :

  3. Tick pada check box aksi untuk nama pengguna yang akan diberikan akses :

    4. Klik tombol OK pada pesan yang muncul kemudian klik tombol Tutup untuk kembali ke form awal.

  4. Lakukan cara yang sama untuk memberikan akses sub menu yang ada pada HELP untuk user tertentu.

    5. Klik pada sub menu Bantuan, akan muncul list data user seperti di bawah ini :

    Tick pada check box aksi untuk nama pengguna yang akan diberikan akses :
