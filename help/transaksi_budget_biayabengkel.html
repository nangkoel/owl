TRANSAKSI BUDGET BIAYA BENGKEL

Modul : Anggaran
Tentang : Transaksi Budget Biaya Bengkel
Isi :

Langkah-langkah input Transaksi Budget Biaya Bengkel :

Masuk ke menu, akan terdapat Form Biaya Bengkel dan 4 Tab yaitu Tab SDM, Tab Material, Tab Peralatan, Tab Lainnya dan Tab Tutup seperti di bawah ini :

  1. Form Biaya Bengkel
    1. Masuk ke menu, isikan tahun budget dan pilih bengkel kemudian klik tombol Simpan. Tipe Anggaran akan terisi otomatis oleh sistem.

  2. Tab SDM
    1. Setelah simpan form biaya bengkel di atas, secara otomatis akan muncul HK Efektif pada Tab SDM. Pilih kode anggaran dan isikan jumlah personel. Total Biaya akan terbentuk secara otomatis oleh sistem kemudian klik tombol Simpan.

    2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

    3. Klik icon untuk menghapus data pada Tab SDM.
  3. Tab Material
    1. Klik Tab Material, pilih kode anggaran, isikan kode barang dan jumlah kemudian klik tombol Simpan. Total Harga akan dihitung oleh sistem.

    2. Untuk mengisi kode barang, klik icon . Berikut tampilan dari form pencarian barang :

      3. Klik pada barang yang dimaksud.

    3. Data yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

    4. Klik icon untuk menghapus data pada Tab Material.
  4. Tab Peralatan
    1. Klik Tab Peralatan, pilih kode anggaran, isikan kode barang dan jumlah kemudian klik tombol Simpan. Total Harga akan dihitung oleh sistem.

    2. Untuk mengisi kode barang, klik icon . Berikut tampilan dari form pencarian barang :

      3. Ketikkan nama barang yang dicari, tekan tombol Find kemudian klik barang yang dimaksud pada hasil pencariannya.

    3. Data yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

    4. Klik icon untuk menghapus data pada Tab Peralatan.
  5. Tab Lainnya
    1. Klik Tab Lainnya, pilih kode anggaran, nomor akun dan isikan total biaya kemudian klik tombol Simpan.

    2. Data yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

    3. Klik icon untuk menghapus data pada Tab Lainnya.
  6. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget. Klik tombol List Data untuk menampilkan List Data budget biaya bengkel :

    2. Klik tombol Tutup untuk melakukan finalisasi budget, akan muncul konfirmasi berikut :

      3. Klik tombol OK jika sudah yakin tidak ada perubahan data pada budget. Konfirmasi ini akan muncul pada setiap baris data budget di atas.

      Jika semua data budget sudah tutup maka akan muncul pesan di bawah ini :

      Klik tombol OK.
