TRANSAKSI BUDGET REGION

Modul : Anggaran
Tentang : Transaksi Budget Region
Isi :

Langkah-langkah input Transaksi Budget Region :

Masuk ke menu, akan terdapat form budget region dan 3 Tab yaitu Tab List Data, Tab Sebaran dan Tab Tutup seperti di bawah ini :

  1. Form Budget Region
    1. Lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data budget region.

  2. Tab List Data
    1. Data yang sudah tersimpan akan muncul pada Tab List Data berikut ini :

    2. Klik icon untuk hapus budget region.
  3. Tab Sebaran
    1. Klik Tab Sebaran, akan muncul semua sebaran budget region :

    2. Klik icon untuk mengisi sebaran per bulan, berikut tampilan form nya :

      3. Sebaran ini akan dibagi rata per bulan, namun dapat disesuaikan dengan mengisi prediksi persen saja, nominal akan otomatis dihitung oleh sistem :

    3. Klik icon untuk simpan data sebaran.
    4. Klik icon untuk mengosongkan data sebaran.
    5. Data sebaran sudah terisi :

  4. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget, pilih tahun budget kemudian klik tombol List Data untuk menampilkan List Data budget region.

    2. Klik tombol Tutup untuk melakukan finalisasi budget, akan muncul konfirmasi berikut :

      3. Klik tombol OK jika sudah yakin tidak ada perubahan data pada budget. Konfirmasi ini akan muncul pada setiap baris data budget di atas.

      Jika semua data budget sudah tutup maka akan muncul pesan di bawah ini :

      Klik tombol OK.
