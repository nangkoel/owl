  1. Summary

    4. Klik pada Total Item untuk melihat List Permintaan Barang dalam format Excel.

    1. Klik icon untuk melihat List Permintaan Pembelian semua PT, berikut tampilannya :

    2. Klik icon untuk melihat List Permintaan Barang semua PT.

    4. Klik pada salah satu PP untuk melihat detail PP, berikut tampilannya :

    6. Klik icon untuk cetak PP dalam format PDF.
  3. List Data
    1. Klik icon untuk melihat list permintaan pembelian yang membutuhkan verifikasi dari Manager Purchasing. Berikut tampilan List PP :

    2. Verifikasi PP dapat dilakukan dengan klik No.PP, akan muncul Form Verivikasi seperti di bawah ini :

    3. Klik icon untuk menambah item barang pada PP, berikut tampilan form nya :

      4. Cari barang yang akan ditambahkan kemudian klik pada barang yang dimaksud :

      Isikan jumlah yang akan dibeli kemudian klik icon untuk menyimpan.

    4. Klik icon untuk mengganti item barang pada pp, berikut tampilan form nya :

      5. Cari barang yang akan menggantikan item barang PP kemudian klik pada barang yang dimaksud maka akan muncul konfirmasi berikut :

      Klik tombol OK untuk mengganti barang.

    5. Klik icon untuk mengajukan kembali PP, berikut tampilan form nya :

      6. Untuk Tolak Beberapa, akan muncul form penolakan dibawah ini :

    6. Klik icon untuk melakukan percakapan dengan Manager Purchasing, berikut tampilan form chat :

      7. Jika sudah terjadi percakapan, maka icon akan berubah menjadi icon .
