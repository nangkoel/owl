SETUP KODE PENGENAAN PAJAK

Modul : Pemasaran
Tentang : Setup Kode Pengenaan Pajak
Isi :

Langkah-langkah Setup Kode Pengenaan Pajak :

  1. Masuk ke menu, isikan kode dan keterangan kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  3. Klik icon untuk edit kode pengenaan pajak.
  4. Klik icon untuk hapus kode pengenaan pajak.
