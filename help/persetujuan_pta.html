PERSETUJUAN PTA

Modul : Anggaran
Tentang : Persetujuan PTA
Isi :

Langkah-langkah Persetujuan PTA :

  1. Masuk ke menu, akan muncul semua list PTA yang perlu persetujuan.
  2. Terdapat fasilitas pencarian berdasarkan pada no. PTA seperti di bawah ini :

    3. Klik icon untuk melihat PTA dalam format PDF.

  3. Jika PTA ditolak, klik tombol Ditolak maka akan muncul form alasan menolak seperti di bawah ini :

    4. Ketikkan alasan ditolaknya PTA kemudian klik tombol Ditolak.

  4. Jika PTA disetujui, klik tombol Disetujui maka akan muncul form persetujuan seperti di bawah ini :

    5. Ketikkan catatan kemudian klik tombol Ajukan untuk mengajukan ke pihak terkait. Akan muncul form persetujuan seperti di bawah ini :

    Pilih nama karyawan dan ketikan keterangan kemudian klik tombol Ajukan.
