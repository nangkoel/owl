LAPORAN PENERIMAAN-PENGELUARAN PER BARANG

Modul : Pengadaan
Tentang : Laporan Penerimaan-Pengeluaran Per Barang
Isi :

Langkah-langkah melihat Laporan Penerimaan-Pengeluaran Per Barang :

  1. Masuk ke menu, pilih unit, periode dan tipe transaksi seperti di bawah ini :

    2. Untuk kode barang, bisa pilih per kode barang atau seluruh barang.

  2. Untuk per kode barang, klik pada field Kode Barang maka akan muncul pencarian barang seperti di bawah ini :

    3. Klik pada kode barang yang diinginkan, kemudian klik tombol Proses maka akan muncul laporan penerimaan dan pengeluaran untuk barang yang dimaksud seperti di bawah ini:

    Laporan ini dapat dibuka dalam format Excel dengan klik icon

  3. Untuk pilih semua kode barang, klik tombol Seluruhnya.

    4. Kemudian klik tombol Proses maka akan muncul laporan penerimaan untuk semua kode barang. Berikut tampilan dari laporan tersebut :

    Laporan ini dapat dibuka dalam format Excel dengan klik icon

    Kolom Tipe Transaksi akan menampilkan angka yang mewakili tipe transaksi yang dipilih dengan keterangan sebagai berikut :

    1. Tipe transaksi mutasi dalam perjalanan = 0
    2. Tipe transaksi penerimaan = 1
    3. Tipe transaksi pengembalian pengeluaran (retur)= 2
    4. Tipe transaksi penerimaan mutasi = 3
    5. Tipe transaksi pengeluaran = 5
    6. Tipe transaksi pengembalian penerimaan = 6
    7. Tipe transaksi pengeluaran mutasi = 7
