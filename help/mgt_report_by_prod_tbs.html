LAPORAN BIAYA PRODUKSI TBS

Modul : Gudang Laporan
Tentang : Laporan Biaya Produksi TBS
Isi :

Langkah-langkah melihat Laporan Biaya Produksi TBS :

Masuk ke menu, akan muncul 2 Tab yaitu Tab Biaya Produksi TBS dan Tab Biaya Produksi TBS per PT seperti di bawah ini :

  1. Tab Biaya Produksi TBS
    1. Pilih perusahaan, unit, afdeling, s/d periode dan Inti/Plasma kemudian klik tombol Preview untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.

    2. Berikut tampilan dari Laporan Biaya Produksi TBS :

  2. Tab Biaya Produksi TBS per PT
    1. Klik Tab Biaya Produksi TBS per PT, pilih s/d periode kemudian klik tombol Preview untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.
