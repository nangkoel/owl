TRANSAKSI PENGGANTIAN TRANSPORT

Modul : SDM
Tentang : Transaksi Penggantian Transport
Isi :

Langkah-langkah input Transaksi Penggantian Transport :

Masuk ke menu, akan terdapat 3 tab yaitu Tab Form, Tab List Data dan Tab Pembayaran seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data.

    2. Saat melakukan proses simpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses.

    3. Isi rincian untuk bahan bakar seperti di bawah ini :

      4. Klik tombol Simpan untuk menyimpan rincian.

    4. Data yang sudah tersimpan akan muncul pada list di bawah ini :

    5. Klik icon untuk hapus data rincian BBM.
  2. Tab List Data
    1. Klik Tab List Data, akan muncul semua list data penggantian transport seperti di bawah ini :

    2. Untuk melihat list data periode tertentu dapat memilih periode yang diinginkan seperti di bawah ini :

    3. Klik icon untuk hapus data penggantian transport.
    4. Klik icon untuk melihat penggantian transport dalam format PDF seperti di bawah ini :

  3. Tab Pembayaran
    1. Klik Tab Pembayaran.

    2. Klik icon untuk menampilkan total biaya pada kolom dibayar.
    3. Cek tanggal bayarnya apakah sudah sesuai.
    4. Klik icon untuk menyimpan pembayaran penggantian transport. Akan muncul konfirmasi seperti di bawah ini :

      5. Klik tombol OK untuk melanjutkan proses penyimpanan.

    5. Data penggantian transport yang sudah dibayar tidak akan muncul lagi pada Tab Pembayaran.
