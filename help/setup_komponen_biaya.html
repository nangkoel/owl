SETUP KOMPONEN BIAYA

Modul : Keuangan
Tentang : Setup Komponen Biaya
Isi :

Langkah-langkah Setup Komponen Biaya :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkannya.

  2. Setelah klik tombol Simpan akan muncul konfirmasi seperti di bawah ini :

    3. Klik tombol OK untuk menyimpan atau klik tombol Cancel untuk membatalkan.

  3. Data yang sudah disimpan akan muncul pada list data berikut :

  4. Klik icon untuk melihat semua komponen biaya dalam format PDF.
  5. Klik icon untuk mencetak komponen biaya.
  6. Klik icon untuk edit data komponen biaya.
  7. Klik icon untuk hapus data komponen biaya.
