TRANSAKSI BUDGET PKS

Modul : Anggaran
Tentang : Transaksi Budget PKS
Isi :

Langkah-langkah input Transaksi Budget PKS :

Masuk ke menu, akan terdapat Form Biaya Pabrik, Form Tutup dan 5 Tab seperti di bawah ini :

  1. Form Biaya Pabrik
    1. Isikan tahun budget, pilih station dan mesin kemudian klik tombol Simpan.

  2. Tab Eksploitasi
    1. Setelah simpan biaya pabrik, lengkapi data pada Tab Eksploitasi.
    2. Klik Tab Eksploitasi, pilih kode anggaran dan isikan jumlah per tahun kemudian klik tombol Simpan seperti di bawah ini :

    3. Data Eksploitasi yang sudah tersimpan akan muncul pada List Data di bawah ini :

    4. Klik icon untuk hapus data eksploitasi.
  3. Tab Material
    1. Klik Tab Material, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data material, akan muncul pesan berikut :

      Klik tombol OK.

    2. Data yang sudah tersimpan akan muncul pada form data tersimpan di bawah ini :

    3. Klik icon untuk hapus data material.
  4. Tab Pemeliharaan
    1. Klik Tab Pemeliharaan, isikan jumlah per tahun kemudian klik tombol Simpan seperti di bawah ini :

    2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

    3. Klik icon untuk hapus data pemeliharaan.
  5. Tab Alat Berat dan Kendaraan
    1. Klik Tab Alat Berat dan Kendaraan, lengkapi data seperti di bawah ini :

      2. Total Biaya akan dihitung oleh sistem. Klik tombol Simpan untuk menyimpan data alat berat dan kendaraan.

    2. Data yang sudah tersimpan akan muncul pada list data di bawah ini :

    3. Klik icon untuk hapus data alat berat dan kendaraan.
  6. Tab Sebaran
    1. Klik Tab Sebaran, akan muncul semua sebaran PKS seperti di bawah ini :

    2. Semua data yang diinput pada Tab harus dibuat sebarannya.
    3. Misalkan akan input sebaran data eksploitasi lembur. Cari mesin SOGM01CG01 dan kode anggaran EXPL-LEMBUR :

    4. Klik icon untuk input sebaran, berikut tampilan form nya :

      5. Jika sebaran rupiah per bulan sudah sesuai, klik icon untuk menyimpan sebaran.

      Klik icon untuk mengosongkan form sebaran.

    5. Data sebaran eksploitasi lembur SOGM01CG01 sudah terisi :

  7. Form Tutup
    1. Setelah selesai mengisi budget maka harus dilakukan finalisasi agar tidak dapat dirubah lagi dan tidak mempengaruhi data yang lain.
    2. Pilih pabrik dan tahun budget kemudian klik tombol Tutup.

    3. Akan muncul konfirmasi di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses. Akan muncul pesan berikut :

      Klik tombol OK.
