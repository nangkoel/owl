BA PELAKSANAAN PEKERJAAN

Modul : Pengadaan
Tentang : Pembuatan BA Pelaksanaan Pekerjaan
Isi :

Langkah-langkah Pembuatan BA Pelaksanaan Pekerjaan :

  1. Masuk ke menu maka akan muncul semua list data kontrak perjanjian kerja.

  2. Klik Edit (tanda pensil) pada kontrak perjanjian kerja di atas yang akan dibuat BA Pelaksanaan Pekerjaan, berikut tampilannya:

  3. Klik pada detail data di atas, maka akan muncul :

    4. Isikan kelengkapan data realisasinya kemudian klik icon untuk menambah detail realisasi.

  4. Saat klik icon maka data sudah tersimpan dan akan muncul baris baru seperti di bawah ini :

    5. Isikan kembali kelengkapan data realisasinya.

  5. Klik icon untuk menyimpan data jika ada perubahan (edit) data.
  6. Klik icon untuk menghapus data.
  7. Klik icon untuk posting data. Jika icon ini masih berwarna merah maka data belum diposting sehingga masih bisa diedit, jika sudah berwarna hijau maka data sudah diposting dan tidak dapat diedit lagi.
  8. Berikut tampilan data yang sudah posting :

    9. Icon menandakan bahwa transaksi sudah posting.
