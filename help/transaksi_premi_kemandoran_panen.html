PENGINPUTAN PREMI KEMANDORAN DAN KERANI PANEN HARIAN

Modul : Kebun
Tentang : Penginputan Premi Kemandoran dan Kerani Panen Harian
Isi :

Langkah-langkah Penginputan Premi Kemandoran dan Kerani Panen Harian :

  1. Masuk ke menu :

    2. Berikut tampilan form nya :

  2. Isi dan selesaikan input data permasing-masing tab sesuai urutannya. Jika ada yang terlewat penginputannya maka Premi mandor atau kerani bisa menghasilkan angka yang salah.

    3. Contoh:
    Hari ini adalah tanggal 2 Juli 2012, maka urutan inputan adalah:
    1. Input premi mandor panen seluruh afdeling untuk tanggal 2 Juli 2012
    2. Input premi mandor kepala seluruh afdeling untuk tanggal 2 Juli 2012
    3. Input premi Kerani Afdeling/Divisi untuk untuk tanggal 2 Juli 2012
    4. Input premi Kerani Panen untuk tanggal 2 Juli 2012

    Berikut contoh input per tab :

    1. Tab Mandor

      2. Pilih Tanggal dan Nama Mandor kemudian klik tombol Simpan seperti di bawah ini :

      Setelah disimpan maka data tersebut akan muncul pada List Data seperti di atas. Klik icon untuk menghapus data.

    2. Tab Mandor 1/Kepala

      3. Pilih Tanggal dan Nama Mandor Kepala kemudian klik tombol Simpan seperti di bawah ini :

      Setelah disimpan maka data tersebut akan muncul pada List Data seperti di atas. Klik icon untuk menghapus data.

    3. Tab Afdeling

      4. Pilih Tanggal dan Nama kemudian klik tombol Simpan seperti di bawah ini :

      Setelah disimpan maka data tersebut akan muncul pada List Data seperti di atas. Klik icon untuk menghapus data.

    4. Tab Panen

      5. Pilih Tanggal dan Nama kemudian klik tombol Simpan seperti di bawah ini :

      Setelah disimpan maka data tersebut akan muncul pada List Data seperti di atas. Klik icon untuk menghapus data.

    5. Tab List Data/Posting

      6. Pilih List Data Periode kemudian klik tombol Lihat untuk menampilkan semua list data atau klik tombol Excel untuk melihat list data dalam format Excel seperti di bawah ini :

      Klik icon untuk melakukan posting data. Saat melakukan posting akan muncul konfirmasi seperti di bawah ini :

      Klik tombol OK untuk posting data, klik tombol Cancel untuk membatalkan proses posting.Setelah diposting data tidak dapat diubah.

      Berikut tampilan List Data setelah dilakukan posting :

  3. Premi otomatis akan ditarik ke gaji, dan akan teralokasi ke biaya secara otomatis.
