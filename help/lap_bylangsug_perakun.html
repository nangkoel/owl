LAPORAN BIAYA LANGSUNG PER AKUN

Modul : Anggaran
Tentang : Laporan Biaya Langsung Per Akun
Isi :

Langkah-langkah melihat Laporan Biaya Langsung Per Akun :

Masuk ke menu, akan muncul 4 Tab yaitu Tab Thn Tnm, Tab Afdeling, Tab Sebaran dan Tab Cost Element seperti di bawah ini :

  1. Tab Thn Tnm
    1. Klik Tab Thn Tnm, pilih tahun budget dan unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau Excel di atas.

    2. Berikut tampilan dari laporan biaya langsung kelompok akun berdasarkan tahun tanam :

    3. Klik pada salah satu baris untuk melihat detail laporan per no akun. Berikut tampilan dari detail nya :

    4. Klik icon untuk melihat detail laporan dalam format Excel.
  2. Tab Afdeling
    1. Klik Tab Afdeling, pilih tahun budget dan unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau Excel di atas.

    2. Berikut tampilan dari laporan biaya langsung kelompok akun berdasarkan afdeling :

    3. Klik pada salah satu baris untuk melihat detail laporan per no akun. Berikut tampilan dari detail nya :

    4. Klik icon untuk melihat detail laporan dalam format Excel.
  3. Tab Sebaran
    1. Klik Tab Sebaran, pilih tahun budget dan unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau Excel di atas.

    2. Berikut tampilan dari laporan biaya langsung berdasarkan sebaran per bulan :
