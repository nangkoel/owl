LAPORAN PENGOBATAN KARYAWAN

Modul : SDM
Tentang : Laporan Pengobatan Karyawan
Isi :

Langkah-langkah melihat Laporan Pengobatan Karyawan yang belum dibayar :

Masuk ke menu, akan terdapat Tab yaitu Tab Rincian, Tab Ranking Diagnosa, Tab Ranking Biaya/Karyawan, dan Tab Ranking Biaya/Diagnosa seperti di bawah ini :

  1. Tab Rincian
    1. Klik Tab Rincian, akan muncul semua rincian pengobatan karyawan.
    2. Terdapat pilihan tahun plafond, kode organisasi dan rumah sakit yang ingin ditampilkan seperti di bawah ini :

    3. Klik icon untuk menampilkan Laporan Pengobatan Karyawan dalam format Excel.
    4. Klik icon untuk menampilkan detail transaksi pengobatan karyawan seperti di bawah ini :


  2. Tab Ranking Diagnosa
    1. Klik Tab Ranking Diagnosa, pilih Tahun Plafond dan Kode Organisasi maka akan muncul ranking diagnosa seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Ranking Diagnosa dalam format Excel.
  3. Tab Ranking Biaya/ Karyawan
    1. Klik Tab Ranking Biaya/ Karyawan, pilih Tahun Plafond dan Kode Organisasi maka akan muncul ranking biaya per karyawan seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Ranking Biaya per Karyawan dalam format Excel.
  4. Tab Ranking Biaya/ Diagnosa
    1. Klik Tab Ranking Biaya/ Diagnosa, pilih Tahun Plafond dan Kode Organisasi maka akan muncul ranking biaya per diagnosa seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Ranking Biaya per Diagnosa dalam format Excel.
