SETUP PERIODE AKUNTANSI

Modul : Setup
Tentang : Setup Periode Akuntansi
Isi :

Langkah-langkah Setup Periode Akuntansi :

  1. Masuk ke menu, lengkapi data kemudian klik tombol Simpan seperti di bawah ini :

    2. Input periode akuntansi ini hanya dilakukan sekali saja, selanjutnya otomatis akan terinput setelah melakukan tutup buku bulanan.

  2. Saat akan simpan periode akuntansi akan muncul konfirmasi seperti di bawah ini :

    3. Klik tombol OK untuk melanjutkan proses.

  3. Data periode akuntansi yang sudah tersimpan akan muncul pada List Data di bawah ini :

    4. Jika belum tutup buku maka pada kolom tutup buku = 0, jika sudah tutup buku maka pada kolom tutup buku = 1.

  4. Klik icon untuk melihat list data periode akuntansi dalam format PDF.
  5. Klik icon untuk mencetak list data periode akuntansi.
  6. Klik icon untuk edit data periode akuntansi.
  7. Klik icon untuk hapus data periode akuntansi.
