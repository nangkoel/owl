Modul : Anggaran
Tentang : Transaksi Produksi Estate
Isi :

Langkah-langkah input Transaksi Produksi Estate :

Masuk ke menu, akan terdapat 2 Form yaitu Form Entry Data dan Form Tutup seperti di bawah ini :

  1. Form Entry Data
    1. Isikan tahun budget, pilih blok dan isikan janjang/pokok/tahun kemudian klik tombol Simpan. Untuk pokok produksi, BJR dan Total (jjg) akan dihitung oleh sistem.

    2. Saat klik tombol Simpan akan muncul Form Sebaran/ Bulan seperti di bawah ini :

      3. Sebaran/ bulan ini dapat disesuaikan jumlahnya dengan mengisikan prediksi persen per bulannya saja, jumlah JJG akan dihitung oleh sistem seperti di bawah ini :

      Klik icon untuk menyimpan sebaran/bulan.

      Klik icon untuk mengosongkan form.

    3. Data produksi kebun yang sudah tersimpan akan muncul pada List Data di bawah ini :

    4. Terdapat fasilitas filter untuk menampilkan List Data yang diinginkan berdasarkan tahun budget dan/ atau kode blok seperti di bawah ini :

    5. Klik icon untuk hapus data produksi kebun.
    6. Klik pada baris data untuk edit data produksi kebun per blok.
  2. Form Tutup
    1. Setelah selesai mengisi data produksi kebun maka harus dilakukan finalisasi untuk budget tersebut agar tidak dapat dirubah lagi dan tidak mempengaruhi data yang lain.
    2. Pilih tahun budget dan unit kemudian klik tombol Tutup.

    3. Akan muncul konfirmasi di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses.
