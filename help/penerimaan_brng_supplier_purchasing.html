Modul : Pengadaan
Tentang : Penerimaan Barang Supplier (Purchasing)
Isi :

Langkah-langkah Penerimaan Barang Supplier (Purchasing) :

Masuk ke menu, pada halaman awal terdapat 2 tab yaitu tab Penerimaan Barang dan List Data.

  1. Penerimaan Barang
    1. Klik icon di atas untuk menampilkan form pencarian PO seperti di bawah ini :

      2. Sebelum melakukan pencarian PO, lakukan langkah b terlebih dahulu tanpa menutup form pencarian tersebut.

    2. Isikan field Mengetahui dan Nama Penerima seperti di bawah ini :

      3. Setelah mengisi Mengetahui dan Nama Penerima kemudian lakukan pencarian PO pada langkah a tadi seperti tampilan di bawah ini :

      Klik pada PO yang dimaksud.

    3. Setelah PO dipilih maka akan muncul no. dokumen dan rincian dari PO tersebut, berikut tampilannya :

      4. Klik tombol Simpan kemudian klik tombol Selesai maka barang pada PO tersebut sudah diterima.

  2. List Data
    1. Klik Tab List Data maka akan muncul form pencarian transaksi penerimaan barang, isikan no transaksi yang dicari kemudian klik tombol Cari.

      2. Berikut tampilan form pencarian dan hasil pencariannya :

    2. Klik icon untuk melakukan penerimaan barang kembali jika masih ada barang yang belum diterima.
    3. Klik icon untuk menghapus transaksi penerimaan barang.
    4. Klik icon untuk melakukan posting transaksi penerimaan barang.
    5. Setelah dilakukan proses posting, tampilan Action akan berubah menjadi icon saja :

      6. Klik icon untuk melihat Bukti Penerimaan Barang seperti di bawah ini :
