PROSES PENGGAJIAN BULANAN UNIT

Modul : SDM
Tentang : Proses Penggajian Bulanan Unit
Isi :

Langkah-langkah melakukan Proses Penggajian Bulanan Unit:

  1. Masuk ke menu, pilih periode gaji kemudian klik tombol List Data untuk memunculkan list data gaji karyawan bulanan unit.

  2. Berikut tampilan list data gaji karyawan bulanan unit :

    3. Klik tombol Proses untuk melakukan posting.

  3. Akan muncul pesan seperti di bawah ini :

    4. Klik tombol OK.
