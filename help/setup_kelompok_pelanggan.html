Langkah-langkah Setup Kelompok Pelanggan :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Untuk pengisian nama akun, klik icon maka akan muncul form seperti di bawah ini :

    3. Ketikkan nama akun yang dicari kemudian klik tombol Find. Pilih akun yang dimaksud pada Hasil Pencarian tersebut.

  3. Data yang sudah tersimpan akan muncul pada daftar di bawah ini :

  4. Klik icon untuk edit data kelompok pelanggan.
  5. Klik icon untuk hapus data kelompok pelanggan.
