TRANSAKSI PERSEDIAAN HASIL

Modul : Pabrik
Tentang : Transaksi Persediaan Hasil
Isi :

Langkah-langkah input Transaksi Persediaan Hasil :

  1. Masuk ke menu, klik icon untuk input transaksi persediaan hasil.

    2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk menyimpan data header.

  2. Data yang sudah tersimpan akan muncul pada List Data atau klik icon untuk melihat semua list data.
  3. Terdapat fasilitas pencarian untuk data persediaan hasil seperti di bawah ini :

    4. Pencarian ini berdasarkan no. transaksi kemudian klik tombol Cari.

  4. Berikut hasil pencariannya :

  5. Klik icon untuk mencetak semua list data persediaan hasil.
  6. Klik icon untuk melihat semua list data persediaan hasil dalam format PDF.
  7. Klik icon untuk edit data persediaan hasil.
  8. Klik icon untuk hapus data persediaan hasil.
