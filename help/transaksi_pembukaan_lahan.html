Isi :

Langkah-langkah Transaksi Pembukaan Lahan:

  1. Masuk ke menu, klik icon untuk membuat transaksi pembukaan lahan.
  2. Berikut tampilan form tambah header :

    3. Lengkapi data seperti di atas kemudian klik tombol Simpan. No. Transaksi akan terbentuk secara otomatis seperti di bawah ini :

  3. Setelah simpan data header maka akan muncul form detail seperti di bawah ini :

    1. Tab Prestasi

      2. Lengkapi data kode kegiatan, kode organisasi, hasil kerja dan jumlah HK kemudian klik tombol Simpan. Setelah simpan detail prestasi maka data tersebut akan muncul pada tabel prestasi di bawah ini :

    2. Tab Kehadiran

      3. Klik Tab Kehadiran. Lengkapi data NIK, kehadiran, jumlah HK dan Premi kemudian klik tombol Simpan. UMR/hari akan muncul secara otomatis. No detail kehadiran juga secara otomatis terbentuk. Setelah simpan detail kehadiran maka data tersebut akan muncul pada tabel absensi di bawah ini :

      Klik icon untuk edit data detail kehadiran.

      Klik icon untuk delete data detail kehadiran.

    3. Tab Material

      4. Klik Tab Material. Lengkapi data Kode Barang, kuantitas dan volume kemudian klik tombol Simpan. Untuk mengisi kode barang, lakukan pencarian barang seperti di bawah ini :

      Klik pada item barang yang dimaksud. Setelah detail kehadiran tersimpan maka data tersebut akan muncul pada tabel pakai material di bawah ini :

      Klik icon untuk edit data detail material.

      Klik icon untuk delete data detail material.

  5. Klik icon untuk meihat semua list data transaksi pembukaan lahan atau lakukan pencarian data seperti di bawah ini :

    6. Klik tombol Cari untuk melakukan pencarian.

  8. Klik icon pada form cetak untuk melihat list transaksi pembukaan lahan dalam format PDF.
  9. Klik icon untuk edit data transaksi pembukaan lahan.
  10. Klik icon untuk menghapus data transaksi pembukaan lahan.
  11. Klik icon untuk posting transaksi pembukaan lahan. Setelah dilakukan posting, icon akan berubah menjadi berwarna hijau
