TRANSAKSI ALOKASI SUPERVISI

Modul : Anggaran
Tentang : Transaksi Alokasi Supervisi
Isi :

Langkah-langkah input Transaksi Alokasi Supervisi :

Masuk ke menu, akan terdapat 4 Tab yaitu Tab Keterangan, Tab HK Supervisi

  1. Tab HK Supervisi
    1. Klik Tab HK Supervisi, isikan tahun budget, upah/hari supervisi dan jumlah personel kemudian klik tombol Preview. Total upah supervisi akan dihitung oleh sistem.

    2. Akan muncul list data HK supervisi dan rupiah supervisi seperti di bawah ini :

      3. HK supervisi dan rupiah supervisi bisa disesuaikan untuk per kegiatannnya. Klik tombol Simpan di atas untuk menyimpan data HK Supervisi.

    3. Akan muncul konfirmasi di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses. Jika proses sudah selesai akan muncul pesan berikut :

      Klik tombol OK.

  2. Tab Sebaran
    1. Klik Tab Sebaran, isikan tahun budget kemudian klik tombol Preview seperti di bawah ini :

    2. Akan muncul list data sebaran seluruh kegiatan seperti di bawah ini :

      3. Klik pada baris kegiatan untuk mengosongkan sebaran. Jika data sebaran sudah sesuai klik tombol Simpan untuk menyimpan sebaran di atas.

    3. Akan muncul konfirmasi di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses. Jika proses sudah selesai akan muncul pesan berikut :

      Klik tombol OK.

  3. Tab Proses Ulang
    1. Klik Tab Proses Ulang, isikan tahun budget kemudian klik tombol Hapus untuk melakukan proses ulang.

    2. Akan muncul konfirmasi berikut :

      3. Klik tombol OK untuk melanjutkan proses.Jika proses sudah selesai akan muncul pesan berikut :

      Klik tombol OK.
