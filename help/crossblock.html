Modul : Kebun
Tentang : Rekam Jejak Planter
Isi :

Masuk ke menu, akan terdapat 2 Tab yaitu Tab Form dan Tab Rekap seperti di bawah ini :

  1. Tab Form
    1. Lengkapi data seperti di bawah ini kemudian klik tombol Simpan :

      2. Form isian akan disesuaikan dengan kelompok kegiatan yang dipilih.

      Tampilan form jika memilih kelompok kegiatan Ancak Panen :

      Tampilan form jika memilih kelompok kegiatan Mutu TPH :

    2. Saat melakukan proses simpan, akan muncul pesan berikut :

      3. Klik tombol OK.

    3. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

    4. Klik icon untuk edit data Rekam Jejak Planter.
    5. Klik icon untuk hapus data Rekam Jejak Planter.
  2. Tab Rekap
    1. Klik Tab Rekap untuk melihat list data Rekam Jejak Planter. Pilih kode organisasi dan periode kemudian klik tombol Preview. Klik tombol Excel untuk melihat rekap Rekam Jejak Planter dalam format Excel.
