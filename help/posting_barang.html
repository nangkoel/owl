POSTING TRANSAKSI

Modul : Pengadaan
Tentang : Posting Transaksi
Isi :

Langkah-langkah posting transaksi :

Masuk ke menu, akan terdapat form daftar gudang dan 2 tab yaitu tab Blm Posting dan Daftar Transaksi.

  1. Form Daftar Gudang

    2. Pilih gudang yang dimaksud kemudian klik Simpan. Tombol Ganti digunakan untuk mengganti pilihan gudang.

  2. Tab Blm Posting
    1. Setelah klik tombol Simpan di atas maka akan muncul semua transaksi yang belum posting baik transaksi penerimaan barang, pemakaian barang atau pengembalian/retur barang. Berikut tampilannya :

    2. Untuk memastikan transaksi yang akan diposting memang belum diposting maka lakukan pencarian seperti di bawah ini :

    3. Berikut hasil dari pencarian di atas :

    4. Pastikan saldo barang tidak nol agar dapat dilakukan proses posting. Klik tombol Proses untuk melakukan proses posting transaksi tersebut. Akan muncul window berikut:

    5. Klik tombol Posting, akan muncul konfirmasi seperti di bawah ini:

    6. Klik OK maka akan muncul pesan berikut:

      7. Klik OK maka transaksi tersebut sudah posting.

  3. Tab Daftar Transaksi
    1. Klik Tab Daftar Transaksi maka akan muncul semua list transaksi yang sudah posting, berikut tampilannya :

    2. Untuk memastikan transaksi sudah posting maka lakukan pencarian seperti di bawah ini :

    3. Berikut hasil pencarian di atas :

    4. Klik icon untuk melihat Bukti Penerimaan Barang dalam format PDF. Status Transaksi Confirmed karena sudah posting.

    5. Jika yang posting adalah transaksi pemakaian barang maka dapat melihat Berita Acara Serah Terima Barang dalam format PDF.

    6. Jika yang posting adalah transaksi pengembalian/retur barang maka dapat melihat Dokumen Retur Barang dalam format PDF.
