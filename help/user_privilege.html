PENGATURAN HAK AKSES

Modul : Administrator
Tentang : Pengaturan Hak Akses
Isi :

Langkah-langkah Pengaturan Hak Akses :

Masuk ke menu akan terdapat form Levelisasi dan Pilihan Hak, form Status seperti di bawah ini :

  • Levelisasi dan Pilihan Hak
    1. Atur Levelisasi digunakan untuk pengaturan level terhadap menu. Berikut tampilan dari Pengaturan Levelisasi :

      2. Level 0 untuk umum, semua bisa mengakses
      Level 1 untuk admin
      Level 5 untuk user

      Setelah selesai mengatur level menu, klik tombol Shahkan kemudian klik tombol Tutup untuk kembali ke menu awal.

    2. Hak-hak Pengguna, Expand pada folder Hak-hak Pengguna maka akan muncul pengaturan hak berdasarkan level dan pengaturan hak per pengguna :

      1. Gunakan Hak berdasarkan Level

        2. Atur akses level per user, klik tombol Shahkan kemudian klik tombol Tutup untuk kembali ke menu awal.

      2. Gunakan Hak-hak Detail Per Pengguna

        3. Klik pada nama pengguna untuk memberikan hak, berikut tampilannya :

        Tick pada check box pada modul yang dibukakan akses nya, kemudian expand modul tersebut untuk memberikan akses pada menu dan submenu :

        Jika sudah selesai memberikan akses menu, klik tombol Done kemudian klik tombol Shahkan.

  • Status

    • Pada status di atas yang aktif adalah tipe akses detail.
