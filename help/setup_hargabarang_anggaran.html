SETUP HARGA BARANG ANGGARAN

Modul : Anggaran
Tentang : Setup Harga Barang Anggaran
Isi :

Langkah-langkah Setup Harga Barang Anggaran :

  1. Masuk ke menu, klik icon untuk input harga barang anggaran. Berikut tampilan dari form input :

    2. Klik tombol Proses untuk menampilkan list data harga barang anggaran.

  2. Berikut tampilan list data harga barang anggaran :

    3. Varian (%) dan Harga Budget di atas dapat disesuaikan. Jika sudah sesuai, klik tombol Simpan di atas.

  3. Saat proses simpan telah selesai, akan muncul pesan di bawah ini :

    4. Klik tombol OK.

  4. Lakukan input kembali untuk kelompok barang yang lain.
  5. Klik icon untuk melihat harga barang anggaran semua regional seperti di bawah ini :

  6. Klik tombol List Data untuk menampilkan list data harga barang anggaran :

  7. Untuk mengubah harga barang anggaran, lengkapi data seperti di bawah ini kemudian klik tombol Simpan :

    8. Untuk mengisi kode barang, klik icon . Berikut tampilan form pencarian barang :

    Ketikkan nama barang yang dimaksud kemudian klik barang pada hasil pencarian di atas.

  8. Klik tombol Hapus untuk menghapus data harga barang anggaran. Akan muncul konfirmasi di bawah ini :

    9. Klik tombol OK untuk melanjutkan proses hapus. Akan muncul pesan di bawah ini :

    Klik tombol OK.

  9. Klik tombol Excel untuk melihat list data harga barang anggaran dalam format Excel.
  10. Klik tombol Ubah untuk edit harga barang anggaran.
  11. Klik tombol Tutup untuk melakukan finalisasi agar data tidak dapat dirubah lagi. Sebelum melakukan finalisasi, klik tombol List Data terlebih dahulu untuk melakukan pengecekan data.

    12. Saat klik tombol Tutup akan muncul konfirmasi di bawah ini :

    Klik tombol OK untuk melanjutkan proses. Akan muncul pesan berikut :

    Klik tombol OK.
