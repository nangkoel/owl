TRANSAKSI BLOK ANGGARAN

Modul : Anggaran
Tentang : Transaksi Blok Anggaran
Isi :

Langkah-langkah input Transaksi Blok Anggaran :

Masuk ke menu, akan terdapat 3 Tab yaitu Tab Blok Lama, Tab Blok Baru dan Tab Tutup seperti di bawah ini :

  1. Tab Blok Lama
    1. Isikan tahun budget dan pilih afdeling kemudian klik tombol Preview.

    2. Akan muncul List Data berikut :


      3. Klik tombol Simpan Seluruhnya untuk menyimpan seluruh budget blok.

    3. Akan muncul konfirmasi seperti di bawah ini :

      4. Klik tombol OK.

    4. Data yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

    5. Klik icon untuk melihat list data blok lama dalam format Excel.
    6. Klik icon untuk edit list data blok lama.
  2. Tab Blok Baru
    1. Klik Tab Blok Baru, isikan tahun budget dan pilih afdeling kemudian klik tombol Preview.

    2. Akan muncul List Data seperti di bawah ini :

      3. Lengkapi data seperti di atas kemudian klik icon untuk menyimpan data blok baru. Akan muncul konfirmasi seperti di bawah ini :

      Klik tombol OK.

    3. Data blok baru yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

    4. Klik icon untuk melihat list data blok baru dalam format Excel.
    5. Klik icon untuk edit list data blok baru.
  3. Tab Tutup
    1. Setelah selesai mengisi blok lama dan blok baru maka harus dilakukan finalisasi untuk budget blok tersebut agar tidak dapat dirubah lagi dan tidak mempengaruhi data yang lain.
    2. Klik Tab Tutup, pilih tahun budget kemudian klik tombol Proses.

    3. Akan muncul konfirmasi seperti di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses. Akan muncul pesan berikut :

      Klik tombol OK.
