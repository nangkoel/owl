TRANSAKSI PREMI PENGAWAS

Modul : Kebun
Tentang : Transaksi Premi Pengawas
Isi :

Langkah-langkah Input Transaksi Premi Pengawas :

  1. Masuk ke menu, lengkapi data pada form premi pengawas seperti di bawah ini :

    2. Klik tombol Simpan untuk menyimpan atau klik tombol Batal untuk membatalkan.

  2. Data premi pengawas yang sudah disimpan akan muncul pada List Data di bawah ini :

    3. Pilih periode yang akan dilihat, maka data premi pengawas periode tersebut akan muncul.

  3. Klik icon untuk menghapus data premi pengawas.
