PERSETUJUAN IJIN/CUTI

Modul : SDM
Tentang : Persetujuan Ijin/Cuti
Isi :

Langkah-langkah Persetujuan Ijin/Cuti oleh Atasan :

  1. Masuk ke menu, pilih nama karyawan yang akan disetujui ijin/cuti nya seperti di bawah ini :

    2. Klik tombol Disetujui untuk menyetujui pengajuan ijin/cuti, klik tombol Ditolak untuk menolak pengajuan ijin/cuti, atau klik tombol Forward untuk forward persetujuan ke karyawan lain.

    Berikut form forward persetujuan cuti :

  2. Data pengajuan ijin/cuti sudah disetujui atasan tetapi masih menunggu persetujuan dari pihak HRD :

  3. Klik icon untuk melihat semua list data ijin/cuti dalam format Excel.
  4. Klik icon untuk cetak Surat Ijin/Cuti dalam format PDF.

Langkah-langkah Persetujuan Ijin/Cuti oleh HRD :

  1. Masuk ke menu, pilih nama karyawan yang akan disetujui ijin/cuti nya seperti di bawah ini :

    2. Klik tombol Disetujui untuk menyetujui pengajuan ijin/cuti atau klik tombol Ditolak untuk menolak pengajuan ijin/cuti.

  2. Data pengajuan ijin/cuti sudah disetujui atasan dan HRD :

  3. Klik icon untuk melihat semua list data ijin/cuti dalam format Excel.
  4. Klik icon untuk cetak Surat Ijin/Cuti dalam format PDF.
