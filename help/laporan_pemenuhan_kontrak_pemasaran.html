LAPORAN PEMENUHAN KONTRAK

Modul : Pemasaran
Tentang : Laporan Pemenuhan Kontrak
Isi :

Langkah-langkah meihat Laporan Pemenuhan Kontrak :

Masuk ke menu, akan terdapat 3 form seperti di bawah ini :

  1. Laporan Pemenuhan Kontrak
    1. Pilih Periode dan Nama Barang kemudian klik tombol Preview untuk menampilkan Laporan :

      2. Laporan ini dapat diakses dalam format PDF dan Excel dengan klik tombol PDF atau tombol Excel.

    2. Berikut tampilan dari Laporan Pemenuhan Kontrak :

    3. Klik pada salah satu kontrak untuk melihat detail laporan seperti di bawah ini :

      4. Detail laporan dapat diakses dalam format PDF dan Excel dengan klik tombol PDF atau Excel di atas.

  2. Kontrak Belum Terpenuhi
    1. Pilih Tahun dan Komoditi kemudian klik tombol Preview untuk menampilkan Laporan dalam format HTML atau klik tombol Excel untuk menampilkan Laporan dalam format Excel :

    2. Berikut tampilan dari Laporan Kontrak Belum Terpenuhi :

  3. Range Pengiriman
    1. Pilih Komoditi, Tgl Dari dan Sampai Tanggal kemudian klik tombol Preview untuk menampilkan Laporan dalam format HTML atau klik tombol Excel untuk menampilkan Laporan dalam format Excel :
