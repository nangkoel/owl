TRANSAKSI TOTAL ALOKASI JAM KENDARAAN

Modul : Anggaran
Tentang : Transaksi Total Alokasi Jam Kendaraan
Isi :

Langkah-langkah input Transaksi Total Alokasi Jam Kendaraan :

  1. Masuk ke menu, isikan tahun budget, pilih kode traksi, kode kendaraan, unit, isikan total jam/tahun kemudian klik tombol Simpan.

  2. Akan muncul Sebaran Bulanan seperti di bawah ini :

    3. Klik icon untuk menyimpan sebaran bulanan alokasi jam kendaraan.

  3. Data yang sudah tersimpan akan muncul pada List Data Tersimpan di bawah ini :

  4. Untuk melakukan edit data sebaran bulanan, klik pada baris yang akan diedit maka akan muncul form edit sebaran di bawah ini :

    5. Klik icon untuk menghapus data total alokasi jam kendaraan.

  5. Terdapat fasilitas filter untuk menampilkan data tersimpan berdasarkan tahun budget, kode kendaraan dan unit seperti di bawah ini :
