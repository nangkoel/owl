Modul : Anggaran
Tentang : Laporan Areal Statement
Isi :

Langkah-langkah melihat Laporan Areal Statement :

  1. Masuk ke menu, pilih tahun budget dan kode organisasi kemudian klik tombol Proses.

  2. Berikut tampilan dari laporan areal statement :

  3. Klik icon untuk melihat laporan dalam format Excel.
  4. Klik icon untuk melihat laporan dalam format PDF.
  5. Klik pada salah satu baris data per afdeling untuk melihat detailnya. Berikut tampilan detail laporan :

    6. Klik icon untuk melihat detail laporan dalam format Excel.
