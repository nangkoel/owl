LAPORAN CURAH HUJAN

Modul : Kebun
Tentang : Laporan Curah Hujan
Isi :

Langkah-langkah Laporan Curah Hujan :

Masuk ke menu, akan muncul 2 Tab seperti di bawah ini :

  1. Tab Curah Hujan Harian
    1. Pilih Afdeling dan Periode kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel.

    2. Berikut tampilan dari Laporan Curah Hujan Harian :

  2. Tab Curah Hujan Bulanan
    1. Pilih Kebun dan Periode kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel.

    2. Berikut tampilan dari Laporan Curah Hujan Bulanan :
