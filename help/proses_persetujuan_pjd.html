PROSES PERSETUJUAN PERJALANAN DINAS

Modul : SDM
Tentang : Proses Persetujuan Perjalanan Dinas
Isi :

Langkah-langkah Proses Persetujuan Perjalanan Dinas oleh Atasan :

  1. Masuk ke menu, akan muncul semua list data perjalanan dinas yang perlu persetujuan.
  2. Terdapat fasilitas pencarian berdasarkan no. transaksi seperti di bawah ini :

    3. Klik tombol Cari untuk melakukan pencarian no. transaksi perjalanan dinas.

  3. Klik icon untuk melihat detail dari transaksi perjalanan dinas tersebut :

    4. Dapat dilakukan penggantian nominal uang muka.

  4. Klik icon untuk menyetujui perjalanan dinas.
  5. Akan muncul konfirmasi seperti di bawah ini :

    6. Klik tombol OK untuk melanjutkan proses.

  6. Perjalanan dinas sudah disetujui oleh atasan tetapi masih menunggu keputusan dari HRD :

  7. Klik icon untuk melihat detail perjalanan dinas. Jika sudah disetujui maka detail uang muka sudah tidak dapat diganti.
  8. Klik icon untuk melihat Surat Perjalanan Dinas dalam format PDF.
