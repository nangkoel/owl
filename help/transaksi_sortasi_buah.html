TRANSAKSI SORTASI BUAH

Modul : Pabrik
Tentang : Transaksi Sortasi Buah
Isi :

Langkah-langkah input Transaksi Sortasi Buah :

  1. Masuk ke menu, klik icon untuk input transaksi sortasi buah. Akan muncul Form Entry Data seperti di bawah ini :

    2. Isikan tanggal kemudian klik tombol Proses maka akan muncul form rincian.

  2. Berikut tampilan form rinciannya :

    3. Lengkapi data seperti di atas kemudian klik icon untuk menyimpan data rincian sortasi buah. Jika rincian sudah selesai diinput, klik tombol Selesai.

  3. Data yang sudah tersimpan akan muncul pada List Data atau klik icon untuk melihat semua list data.
  4. Terdapat fasilitas pencarian untuk data sortasi buah seperti di bawah ini :

    5. Isikan No. Tiket kemudian klik tombol Cari untuk melakukan pencarian.

  5. Berikut hasil pencariannya :

    6. Klik pada salah satu kolom untuk melakukan edit data. Klik icon untuk hapus data rincian sortasi buah.
