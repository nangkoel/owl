LAPORAN ANGGARAN PABRIK

Modul : Anggaran
Tentang : Laporan Anggaran Pabrik
Isi :

Langkah-langkah melihat Laporan Anggaran Pabrik :

  1. Masuk ke menu, pilih anggaran tahunan, kode organisasi dan jenis biaya (Langsung, Umum atau Langsung dan Umum) kemudian klik tombol Simpan untuk menampilkan laporan.

  2. Berikut tampilan dari Laporan Anggaran Pabrik :

  3. Klik icon untuk melihat laporan dalam format Excel.
  4. Klik icon untuk melihat laporan dalam format PDF.
