LAPORAN RIWAYAT SISIPAN

Modul : Kebun
Tentang : Laporan Riwayat Sisipan
Isi :

Langkah-langkah Laporan Riwayat Sisipan :

  1. Masuk ke menu, pilih unit, tanggal dari dan tanggal sampai kemudian klik tombol Preview untuk melihat laporan.

    2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

  2. Berikut tampilan dari Riwayat Sisipan :

  3. Jika ada sisipan maka klik tombol Edit untuk mengisi Jumlah Pokok dan Alasan seperti di bawah ini :

    4. Klik tombol Simpan untuk menyimpan sisipan.

  4. Pada Print Area di bawah ini, transaksi yang memiliki keterangan pokok dan alasan adalah blok yang memiliki sisipan. Jika tidak ada keterangan pokok dan alasan berarti pada blok tersebut tidak terdapat sisipan.
