TRANSAKSI PERMINTAAN TAMBAHAN ANGGARAN

Modul : Anggaran
Tentang : Transaksi Permintaan Tambahan Anggaran
Isi :

Langkah-langkah input Transaksi Permintaan Tambahan Anggaran :

Masuk ke menu, akan terdapat 2 Tab yaitu Tab Buat PTA dan Tab Daftar PTA seperti di bawah ini :

  1. Tab Buat PTA
    1. Lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan permintaan tambahan anggaran.

    2. Akan muncul pesan di bawah ini :

      3. Klik tombol OK.

    3. Data yang sudah tersimpan akan muncul pada list detail di bawah ini :

    4. Klik icon untuk hapus detail data.
    5. Untuk mengajukan PTA, klik tombol Ajukan di atas. Berikut tampilan form pengajuan :

      6. Pilih nama karyawan dan ketikan keterangan kemudian klik tombol Ajukan.

  2. Tab Daftar PTA
    1. Klik Tab Daftar PTA, akan muncul daftar PTA yang ada.

    2. Klik icon untuk melihat PTA dalam format PDF.
    3. Klik icon untuk edit PTA jika belum ada persetujuan.
    4. Klik icon untuk hapus PTA jika belum ada persetujuan.
