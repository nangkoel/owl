PEMBUATAN PENAWARAN HARGA

Modul : Pengadaan
Tentang : Pembuatan Penawaran Harga
Isi :

Langkah-langkah Pembuatan Penawaran Harga :

  1. Buat Baru
    1. Klik icon untuk membuat penawaran harga. Berikut tampilan form penawaran harga :

    2. Klik icon untuk melakukan pencarian No. PP, berikut tampilan form pencarian PP :

      3. Klik pada PP yang dimaksud.

    3. Tampilan setelah memilih PP yang dimaksud :

      4. Klik tombol Proses, akan muncul Daftar Barang di bawah ini :

      Tick pada check box di atas kemudian klik tombol Lanjut, akan muncul form pengisian data supplier seperti di bawah ini :

      Klik pada icon untuk melakukan pencarian supplier, berikut tampilan form pencarian supplier :

      Klik pada supplier yang dimaksud, kemudian klik tombol Add Data di bawah ini :

      Data yang sudah tersimpan akan muncul pada List Data. Klik icon untuk hapus data supplier.

    4. Klik tombol Selesai, maka akan muncul form penawaran harga berikut :

      5. Lengkapi data seperti di atas kemudian klik tombol Simpan.

      Klik tombol Excel untuk melihat Penawaran Harga dalam format Excel.

  2. Ubah
    1. Klik icon untuk ubah penawaran harga.

  3. List Data
    1. Klik icon untuk melihat list data penawaran harga :

    2. Klik icon untuk hapus penawaran harga.
    3. Klik icon untuk cetak penawaran harga dalam format PDF.
    4. Klik icon untuk melihat penawaran harga dalam format Excel.
  4. Fasilitas Pencarian
    1. Pencarian berdasarkan No.PP, No.Penawaran Harga, nama barang dan tanggal kemudian klik tombol Cari :

    2. Berikut hasil pencariannya :

    3. Klik icon untuk edit penawaran harga.
