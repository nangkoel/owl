TRANSAKSI PENGOPERASIAN PABRIK

Modul : Pabrik
Tentang : Transaksi Pengoperasian Pabrik
Isi :

Langkah-langkah input Transaksi Pengoperasian Pabrik :

  1. Masuk ke menu, klik icon untuk input transaksi pengoperasian pabrik. Berikut tampilan form tambah header :

    2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk menyimpan data header.

  2. Setelah data header tersimpan maka akan muncul form tambah detail seperti di bawah ini :

    3. Lengkapi data detail seperti di atas kemudian klik tombol Simpan untuk menyimpan data detail.

  3. Data detail yang sudah tersimpan akan muncul pada Tabel Mesin di bawah ini :

  4. Klik icon untuk edit data detail mesin.
  5. Klik icon untuk hapus data detail mesin.
  6. Klik icon untuk menambahkan material pada detail mesin tersebut. Berikut tampilan dari form tambah material :

    7. Klik tombol Cari di atas maka akan muncul form pencarian barang seperti di bawah ini :

    Ketikkan nama barang yang dicari kemudian klik icon , pilih barang yang dimaksud pada hasil pencariannya.

    Isikan jumlah material tersebut kemudian klik tombol Simpan untuk menyimpan material. Material yang sudah tersimpan akan muncul pada Tabel Material seperti di bawah ini :

    Klik icon untuk edit data material.
    Klik icon untuk hapus data material.

  7. Klik icon untuk melihat semua list data pengolahan pabrik.
  8. Terdapat fasilitas pencarian untuk data pengolahan pabrik seperti di bawah ini :

    9. Isikan no.pengolahan kemudian klik tombol Cari.

  9. Berikut hasil pencariannya :

  10. Pada form cetak, klik icon untuk membuka semua list data dalam format PDF atau klik icon untuk mencetak semua list data.
  11. Pada form List Data, klik icon untuk edit data pengolahan pabrik.
  12. Klik icon untuk hapus data pengolahan pabrik.
  13. Klik icon untuk melihat data pengolahan pabrik dalam format PDF seperti di bawah ini :
