EDIT PEMAKAIAN BARANG

Modul : Pengadaan
Tentang : Edit Pemakaian Barang
Isi :

Langkah-langkah edit pemakaian barang:

1. Masuk ke menu kemudian isikan Gudang dan PT pemilik barang, klik tombol Simpan maka akan muncul tampilan seperti dibawah ini:

2. Klik Tab List Data kemudian cari transaksi yang akan diedit.

3. Hasil pencarian no. transaksi:

4. Klik icon untuk edit pemakaian barang, berikut tampilannya:

5. Klik icon untuk edit rincian data pengeluaran barang, berikut tampilannya:

Pada Rincian pengeluaran barang, yang bisa diedit hanya pada jumlah barang dan kegiatannya.

Misalkan edit jumlah barang menjadi 4 kemudian klik tombol Simpan. Jika sudah tidak ada lagi data yang perlu diedit maka klik tombol Selesai.
