LAPORAN PERSEDIAAN FISIK (STOCK)

Modul : Pengadaan
Tentang : Persediaan Fisik
Isi :

Langkah-langkah melihat laporan persediaan fisik (stock) :

Masuk ke menu, akan muncul 3 form laporan stock seperti di bawah ini :

  1. Laporan Stok Per Perusahaan
    1. Pilih Perusahaan dan Periode kemudian klik tombol Proses untuk menampilkan laporan :

    2. Berikut tampilan dari Laporan Stok Per Perusahaan :

      3. Laporan ini dapat dilihat dalam format Excel dan PDF dengan klik icon .

    3. Untuk melihat detail per barang, klik pada baris barang yang akan dilihat detail nya dalam format PDF seperti di bawah ini :

  2. Laporan Stok Per Gudang
    1. Pilih Perusahaan, Gudang dan Periode kemudian klik tombol Proses untuk menampilkan laporan :

    2. Berikut tampilan dari Laporan Stok Per Gudang :

      3. Laporan ini dapat dilihat dalam format Excel dan PDF dengan klik icon .

    3. Untuk melihat detail per barang, klik pada baris barang yang akan dilihat detail nya dalam format PDF seperti di bawah ini :

  3. Laporan Stok Per Unit
    1. Pilih Unit dan Periode kemudian klik tombol Proses untuk menampilkan laporan :

    2. Berikut tampilan dari Laporan Stok Per Unit :

      3. Laporan ini dapat dilihat dalam format Excel dan PDF dengan klik icon .

    3. Untuk melihat detail per barang, klik pada baris barang yang akan dilihat detail nya dalam format PDF seperti di bawah ini :
