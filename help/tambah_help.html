Langkah-langkah Tambah Help :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Sebelumnya sudah membuat file dalam format HTML yang berisi panduan menu. Jika tidak ada file HTML dan hanya informasi saja maka pada field HTML di atas diisi dengan help/null.

  2. Setelah klik tombol Simpan di atas maka akan muncul pesan berikut :

    3. Pesan ini menunjukkan bahwa data sudah tersimpan. Klik tombol OK.

  3. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  4. Terdapat fasilitas pencarian berdasarkan index, modul dan tentang.
  5. Klik icon untuk edit data help.
  6. Klik icon untuk dislpay data help.
  7. Klik icon untuk delete data help.
