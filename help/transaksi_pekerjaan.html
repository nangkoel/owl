TRANSAKSI PEKERJAAN

Modul : Traksi
Tentang : Transaksi Pekerjaan
Isi :

Langkah-langkah input Transaksi Pekerjaan :

Masuk ke menu, akan terdapat 3 tab yaitu tab Kop, tab Detail Pekerjaan, dan tab Detail Operator.

  1. Tab Kop
    1. Klik tab Kop kemudian lengkapi data seperti di bawah ini :

      2. No. Transaksi akan terbentuk secara otomatis oleh sistem. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

    2. Klik tombol Selesai jika tab Detail Pekerjaan dan tab Detail Operator sudah selesai diisi.
  2. Tab Detail Pekerjaan
    1. Klik tab Detail Pekerjaan kemudian lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data Detail Pekerjaan atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    3. Klik icon untuk edit data detail pekerjaan.
    4. Klik icon untuk hapus data detail pekerjaan.
  3. Tab Detail Operator
    1. Klik tab Detail Operator kemudian lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data detail operator atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    3. Klik icon untuk hapus data detail operator.
  4. Jika data detail pekerjaan dan data operator sudah selesai diisi, lakukan langkah 1b.
