PROSES PERHITUNGAN HARGA AKHIR BULAN

Modul : Pengadaan
Tentang : Proses Perhitungan Harga Akhir Bulan
Isi :

Langkah-langkah Proses Perhitungan Harga Akhir Bulan :

  1. Masuk ke menu, pilih PT Pemilik Barang dan periode kemudian klik tombol Simpan untuk menampilkan Daftar Proses yang berisi semua list barang yang ada di gudang. Jika sudah klik tombol Simpan dan ingin mengganti pilihan PT Pemilik Barang maka klik tombol Ganti untuk mengganti pilihan gudang tersebut.

  2. Berikut tampilan dari Daftar Proses :

  3. Klik tombol Proses di atas maka akan muncul konfirmasi seperti di bawah ini :

    4. Klik tombol OK untuk melakukan proses perhitungan harga akhir bulan atau klik tombol Cancel untuk membatalkannya.

  4. Setelah klik tombol OK dan proses perhitungan harga akhir bulan telah selesai maka akan muncul pesan seperti di bawah ini :

    5. Klik tombol OK.
