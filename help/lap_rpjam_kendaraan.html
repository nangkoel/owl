LAPORAN Rp/ JAM KENDARAAN

Modul : Anggaran
Tentang : Laporan Rp/Jam Kendaraan
Isi :

Langkah-langkah melihat Laporan Rp/Jam Kendaraan :

  1. Masuk ke menu, pilih Tahun Budget dan Kode Traksi kemudian klik tombol Preview untuk menampilkan laporan.

    2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau tombol Excel di atas.

  2. Berikut tampilan dari Laporan Rp/Jam Kendaraan :

  3. Klik tombol Alokasi untuk melihat detail alokasi. Berikut tampilan dari Detail Alokasi :

    4. Klik icon untuk melihat detail alokasi dalam format Excel. Klik icon untuk melihat detail alokasi dalam format PDF.

  4. Klik tombol Biaya Rinci untuk melihat Biaya Rinci seperti di bawah ini :

    5. Klik icon untuk melihat biaya rinci dalam format Excel. Klik icon untuk melihat biaya rinci dalam format PDF.
