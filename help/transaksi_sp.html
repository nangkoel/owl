TRANSAKSI SURAT PERINGATAN

Modul : SDM
Tentang : Transaksi Surat Peringatan
Isi :

Langkah-langkah input Transaksi Surat Peringatan :

Masuk ke menu, akan terdapat 2 Tab yaitu Tab Form dan Tab List Data seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data.

    2. Akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses. Akan muncul pesan seperti di bawah ini :

      Klik tombol OK.

  2. Tab List Data
    1. Klik Tab List Data, klik tombol Cari untuk menampilkan semua list data surat peringatan.

    2. Terdapat fasilitas pencarian berdasarkan no.transaksi seperti di bawah ini :

    3. Klik icon untuk melihat Surat Peringatan dalam format PDF.
    4. Klik icon untuk hapus data surat peringatan.
    5. Klik icon untuk edit data surat peringatan.
