SETUP MASTER BARANG

Modul : Pengadaan
Tentang : Setup Master Barang
Isi :

Langkah-langkah Setup Master Barang :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Kode barang akan terbentuk secara otomatis oleh sistem. No. Kartu bin adalah nomor kartu stock. Klik tombol Simpan untuk menyimpan master barang atau klik tombol Batal untuk membatalkan.

  2. Setelah klik tombol Simpan maka akan muncul konfirmasi seperti di bawah ini :

    3. Klik tombol OK untuk menyimpan atau klik tombol Cancel untuk membatalkan. Saat klik tombol OK akan muncul pesan berikut :

    Klik tombol OK dan data master barang sudah tersimpan.

  3. Terdapat pencarian master barang, berikut tampilan form nya :

    4. Pencarian ini berdasarkan nama barang dan kategori kelompok barang. Klik tombol Cari untuk memulai proses pencarian barang.

  4. Klik icon untuk melihat list master barang dalam format PDF berdasarkan kategori kelompok barang yang dipilih.
  5. Klik icon untuk mencetak master barang.
  6. Tick pada kolom Tidak Aktif untuk menonaktifkan master barang.
  7. Klik icon untuk update detail spec dan foto master barang. Berikut tampilan form detail :

    8. Isikan spec barang dan foto tampak depan, samping, dan atas dengan ukuran file foto maksimal 100 Kb kemudian klik tombol Save maka akan muncul pesan berikut :

    Tutup saja pesan ini dengan klik tanda silang di pojok kanan atas.

  8. Setelah isi detail barang maka akan muncul icon untuk melihat detail master barang. Berikut tampilan detail nya :

  9. Klik icon untuk edit master barang.
  10. Klik icon untuk hapus master barang.
