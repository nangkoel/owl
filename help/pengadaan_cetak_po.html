CETAK ORDER PEMBELIAN (PURCHASE ORDER)

Modul : Pengadaan
Tentang : Cetak Purchase Order (PO)
Isi :

Langkah-langkah Cetak Purchase Order :

  1. Masuk ke menu maka akan muncul semua list order pembelian atau bisa menggunakan pencarian No. PO yang dimaksud.

  2. Klik Cetak maka akan tampil PO dalam bentuk PDF, klik Print untuk mencetak PO tersebut.
