SETUP GAJI POKOK UNIT

Modul : SDM
Tentang : Setup Gaji Pokok Unit
Isi :

Langkah-langkah Setup Gaji Pokok Unit :

  1. Masuk ke menu, masukkan tahun, pilih ID karyawan, ID komponen dan isikan jumlah kemudian klik tombol Simpan.

  2. Data yang sudah tersimpan akan masuk ke List Data Gaji Pokok di bawah ini :

  3. Klik icon untuk melihat list data gaji pokok unit dalam format PDF.
  4. Klik icon untuk cetak list data.
  5. Klik icon untuk melihat list data gaji pokok unit dalam format Excel.
  6. Klik icon untuk edit data gaji pokok karyawan unit.
  7. Klik icon untuk hapus data gaji pokok karyawan unit.
  8. Terdapat fasilitas copy gaji pokok dari konfigurasi gaji tahun tertentu ke tahun tertentu seperti di bawah ini :

    9. Klik tombol Proses maka gaji pokok tahun 2011 akan ter-copy ke gaji pokok tahun 2012.
