LAPORAN BIAYA TBM

Modul : Gudang Laporan
Tentang : Laporan Biaya TBM
Isi :

Langkah-langkah melihat Laporan Biaya TBM :

Masuk ke menu, akan muncul 2 Tab yaitu Tab Biaya TBM Summary dan Tab Biaya TBM Rincian seperti di bawah ini :

  1. Tab Biaya TBM Summary
    1. Pilih perusahaan, unit dan s/d periode kemudian klik tombol Preview untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.

  2. Tab Biaya TBM Rincian
    1. Klik Tab Biaya TBM Rincian, pilih perusahaan, unit dan s/d periode kemudian klik tombol Preview untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.
