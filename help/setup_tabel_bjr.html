Isi :

Langkah-langkah Setup Tabel BJR :

  1. Masuk ke menu maka akan muncul form seperti di bawah ini :

    2. List Data di atas menunjukkan bahwa terdapat 2 tahun produksi yaitu tahun produksi 2011 dan 2012.

  2. Masukkan tahun produksi kemudian klik tombol Ok.

  3. Setelah klik tombol Ok maka akan muncul semua data BJR dengan tahun produksi 2012 seperti di bawah ini :

  4. Untuk input data BJR, lengkapi data seperti di bawah ini :

    5. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  5. Data yang sudah disimpan akan muncul pada List Data di bawah ini :

  6. Klik icon untuk edit data tabel BJR.
  7. Klik icon untuk hapus data tabel BJR.
