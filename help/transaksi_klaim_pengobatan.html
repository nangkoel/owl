Langkah-langkah input Transaksi Klaim Pengobatan :

Masuk ke menu, akan terdapat 3 tab yaitu tab Form, tab Obat-Obatan dan tab List Data seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form, lengkapi data seperti di bawah ini :

      2. No. Transaksi akan terbentuk secara otomatis oleh sistem.

    2. Saat akan menyimpan klaim pengobatan, akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses penyimpanan. Kemudian akan muncul pesan seperti di bawah ini :

      Klik tombol OK.

  2. Tab Obat-Obatan
    1. Setelah data pada tab Form disimpan maka akan secara otomatis menuju ke tab Obat-Obatan.

      2. Isikan nama obat kemudian klik tombol Simpan untuk menyimpan obat. Akan muncul pesan seperti di bawah ini :

      Klik tombol OK.

    2. Data Obat yang sudah tersimpan akan muncul pada List Data di bawah ini :

    3. Klik icon untuk hapus obat.
  3. Tab List Data
    1. Klik Tab List Data maka akan muncul semua list klaim pengobatan seperti di bawah ini :

    2. Klik icon untuk hapus klaim pengobatan, jika klaim sudah dibayar maka data tidak dapat dihapus.
    3. Klik icon untuk melihat detail klaim pengobatan yang belum dibayar seperti di bawah ini :


      4. Tanggal pada Rumah Sakit/Klinik dan Plafon RWJLN masih kosong. Jika sudah dilakukan pembayaran maka tampilan list data nya menjadi seperti di bawah ini :

      Data sudah tidak dapat dihapus. Berikut tampilan detail dari klaim pengobatan yang sudah dibayar :


      Tanggal pada Rumah Sakit/Klinik dan Plafon RWJLN berisi tanggal dan nominal klaim.
