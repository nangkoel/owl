LAPORAN Rp/ JAM BENGKEL

Modul : Anggaran
Tentang : Laporan Rp/Jam Bengkel
Isi :

Langkah-langkah melihat Laporan Rp/Jam Bengkel :

  1. Masuk ke menu, pilih Tahun Budget dan Bengkel kemudian klik tombol Preview untuk menampilkan laporan.

    2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau tombol Excel di atas.

  2. Berikut tampilan dari Summary Laporan Rp/Jam Bengkel :

    3. Klik pada baris di atas untuk melihat Detail Laporan Rp/ Jam Bengkel.

  3. Berikut tampilan dari Detail Laporan Rp/ Jam Bengkel :

  4. Klik icon untuk melihat detail laporan dalam format Excel.
  5. Klik icon untuk melihat detail laporan dalam format PDF.
  6. Klik icon untuk tutup detail laporan.
