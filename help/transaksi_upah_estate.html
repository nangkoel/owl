TRANSAKSI UPAH ESTATE

Modul : Anggaran
Tentang : Transaksi Upah Estate
Isi :

Langkah-langkah input Transaksi Upah Estate :

Masuk ke menu, akan terdapat 2 Tab yaitu Tab Upah(Rp) dan Tab Tutup seperti di bawah ini :

  1. Tab Upah (Rp)
    1. Isikan tahun budget dan pilih kode organisasi kemudian klik tombol Proses.

    2. Akan muncul List Data Budget Upah Estate di bawah ini :

      3. Isikan nominal budget upah per golongan kemudian klik tombol Simpan pada pojok kiri atas untuk menyimpan seluruh data atau Klik tombol Simpan per golongan untuk menyimpan budget upah per golongan.

    3. Saat proses simpan upah selesai, akan muncul pesan seperti di bawah ini :

      4. Klik tombol OK.

  2. Tab Tutup
    1. Setelah selesai mengisi budget upah maka harus dilakukan finalisasi untuk budget tersebut agar tidak dapat dirubah lagi dan tidak mempengaruhi data yang lain.
    2. Klik Tab Tutup, pilih tahun budget kemudian klik tombol Proses.

    3. Akan muncul List Data Tersimpan budget upah 2012 seperti di bawah ini :

      4. Klik tombol Tutup untuk melakukan finalisasi budget upah sehingga tidak dapat dilakukan perubahan lagi.

    4. Akan muncul konfirmasi seperti di bawah ini :

      5. Klik tombol OK untuk melanjutkan proses, setelah proses selesai akan muncul pesan di bawah ini :

      Klik tombol OK.
