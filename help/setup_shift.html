SETUP SHIFT

Modul : Pabrik
Tentang : Setup Shift
Isi :

Langkah-langkah Setup Shift :

  1. Masuk ke menu, pilih kode organisasi, isi shift, pilih mandor dan asisten kemudian klik tombol Simpan untuk menyimpan data shift.

  2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  3. Klik icon untuk membuka list data shift dalam formta PDF.
  4. Klik icon untuk mencetak list data shift.
  5. Klik icon untuk edit data shift.
  6. Klik icon untuk hapus data shift.
  7. Klik icon untuk menambahkan anggota shift. Berikut tampilannya form nya :

    8. Pilih nama anggota kemudian klik tombol Simpan untuk menyimpan data anggota shift. Data anggota shift yang sudah tersimpan akan muncul pada Tabel Anggota Shift.

  8. Klik icon untuk edit data anggota shift.
  9. Klik icon untuk hapus data anggota shift.
