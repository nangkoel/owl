PEMBAYARAN GAJI SEBELUM MASA PENGGAJIAN

Tanya :

Saya mau menanyakan perihal input data di OWL atas pembayaran gaji karyawan sebelum masa penggajian (remise 2).
Misalnya : Gaji atas nama "XXX" per tgl 22 April 2013 :
HK = Rp. 700.600 + Premi = Rp. 40.000,-
Jumlah dibayarkan via KAS (by. Kasir) = Rp. 740.600,-
Opsi saya :
1. a. - untuk JURNAL KAS (by KASIR):
Debet (Gaji YMHD) = 740.600,-
Kredit ( KAS ) = 740.600,-
===== A T A U ====
Debet (Pinjaman Karyawan / Piutang Non Staf) = 740.600,-
Kredit ( KAS ) = 740.600,-

2. b. - untuk INPUT Payroll di OWL (by. Payroll) pada Menu SDM -> Transaksi -> Administrasi Personalia
--> Potongan HK = 700.600
--> Potongan Premi Sudah Dibayar = 40.000

=== ATAU ====

- Potongan Karyawan = 740.600,-

Demikian pak. Mohon saran petunjuknya atas opsi tersebut diatas atau jika ada opsi lain.

Jawab :

Gunakan(Keuangan):

Debet (Pinjaman Karyawan / Piutang Non Staf) = 740.600,-
Kredit ( KAS ) = 740.600,-

dan di SDM owl gunakan
- Potongan Karyawan = 740.600,-
