PENERIMAAN BARANG DARI SUPPLIER (GUDANG)

Modul : Pengadaan
Tentang : Penerimaan Barang dari Supplier (Gudang)
Isi :

Langkah-langkah penerimaan barang dari supplier (gudang) :

Masuk ke menu, akan terdapat form daftar gudang dan 2 tab yaitu tab Penerimaan Barang dan List Data.

  1. Form Daftar Gudang

    2. Pilih gudang yang dimaksud kemudian klik Simpan. Tombol Ganti digunakan untuk mengganti pilihan gudang.

  2. Tab Penerimaan Barang
    1. Setelah klik tombol Simpan di atas maka secara otomatis No. Dokumen penerimaan barang akan terbentuk kemudian isikan Surat Jalan dan Invoice/Faktur seperti di bawah ini :

    2. Klik icon untuk melakukan pencarian No.PO, tampilannya seperti di bawah ini :

    3. Klik pada PO yang dimaksud maka akan muncul Supplier, No. PO dan detail dari PO tersebut :

      4. Pastikan jumlah barang yang ada di PO sesuai dengan jumlah barang yang diterima, jika sudah sesuai maka klik tombol Simpan untuk melakukan penerimaan barang.

    4. Klik tombol Selesai jika sudah selesai melakukan penerimaan per item barang.
  3. Tab List Data
    1. Klik Tab List Data maka akan muncul semua transaksi penerimaan barang di gudang. Lakukan pencarian untuk transaksi penerimaan barang yang dimaksud seperti tampilan di bawah ini :

    2. Berikut hasil pencarian transaksi di atas :

    3. Klik icon untuk melakukan penerimaan barang kembali jika masih ada barang yang belum diterima dan status nya masih belum posting.
    4. Klik icon untuk menghapus transaksi penerimaan barang jika statusnya masih belum posting.
    5. Klik icon untuk melihat bukti penerimaan barang dalam format PDF.

      6. Karena penerimaan ini masih belum diposting maka status bukti penerimaan barang ini masih "Not Confirm".
