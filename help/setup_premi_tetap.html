SETUP PREMI TETAP

Modul : SDM
Tentang : Setup Premi Tetap
Isi :

Langkah-langkah Setup Premi Tetap :

  1. Premi Tetap

    2. Premi hanya diberikan berdasarkan kehadiran, jumlah jam kerja dan atau output yang dihasilkan, baik output berupa hasil kerja maupun terkait waktu kerja (7 jam).
    Apabila karyawan cuti, sakit, ijin, mangkir, pulang cepat/ datang terlambat, pekerja tidak melaksanakan instruksi atasan, maka premi pada hari tersebut tidak dibayarkan.

    1. Masuk ke menu, pilih tipe transaksi premi tetap, pilih jabatan dan isikan nilai premi kemudian klik tombol Simpan seperti di bawah ini :

    2. Untuk pencarian jabatan/ tipe karyawan, klik icon , akan muncul form pencarian seperti di bawah ini :

    3. Data yang sudah tersimpan akan muncul pada list data di bawah ini :

      4. Pilih tipe transaksi premi tetap untuk menampilkan data premi tetap.

    4. Klik icon untuk edit nilai premi tetap.
  2. Insentif

    3. Insentif diberikan kepada karyawan yang bekerja pada hari libur

    1. Masuk ke menu, pilih tipe transaksi insentif, pilih tipe karyawan dan isikan nilai insentif kemudian klik tombol Simpan seperti di bawah ini :

    2. Data yang sudah tersimpan akan muncul pada list data di bawah ini :

      3. Pilih tipe transaksi insentif untuk menampilkan data insentif.

    3. Klik icon untuk edit nilai insentif.
