LAPORAN HARGA BARANG

Modul : Anggaran
Tentang : Laporan Harga Barang
Isi :

Langkah-langkah melihat Laporan Harga Barang :

Masuk ke menu, akan muncul 2 Tab yaitu Tab per Kelompok Barang dan Tab Cari Barang seperti di bawah ini :

  1. Tab per Kelompok Barang
    1. Pilih Tahun Budget, Regional dan Kelompok Barang kemudian klik tombol Proses untuk menampilkan list data barang.

    2. Berikut tampilan dari List Data Barang :

    3. Klik icon untuk melihat list data barang dalam format Excel.
    4. Klik icon untuk melihat list data barang dalam format PDF.
  2. Tab Cari Barang
    1. Klik Tab Cari Barang, pilih Tahun Budget, Regional dan ketikkan nama barang yang akan dicari kemudian klik tombol Proses untuk menampilkan hasil pencarian barang.

    2. Berikut List Data hasil dari pencarian di atas :

    3. Klik icon untuk melihat list data barang dalam format Excel.
    4. Klik icon untuk melihat list data barang dalam format PDF.
