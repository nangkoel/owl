Tentang : Input Blok
Isi :

Langkah-langkah input blok :

1. Masuk ke menu maka akan muncul form pencarian data seperti di bawah ini:

Setelah memilih Kebun dan Afdeling kemudian klik tombol Cari, maka akan muncul form pengisian blok dan list blok yang sudah ada.

2. Berikut tampilan form pengisian blok:

Agar blok bisa muncul di kas bank atau jurnal, cukup medaftarkan blok tersebut dengan memilih kode organisasi, mengisikan tanggal kemudian isikan Tahun Tanam, Tahun Panen dan data yang lain dengan 0. Hal ini dilakukan hanya agar blok tersebut terdaftar saja, untuk kelengkapan datanya bisa dilakukan edit data blok tersebut.

3. Setelah klik Simpan, maka data akan muncul pada list data di bawah ini:

List data blok bisa dilihat dalam format PDF dan bisa langsung dicetak.

Untuk lakukan edit data blok, klik icon kemudian untuk hapus data blok, klik icon
