TRANSAKSI KAS BANK

Modul : Keuangan
Tentang : Transaksi Kas Bank
Isi :

Langkah-langkah Transaksi Kas Bank :

  1. Masuk ke menu, klik icon untuk membuat transaksi kas bank maka akan muncul form tambah header.
  2. Berikut tampilan form tambah header :

    3. Lengkapi data header kemudian klik tombol Simpan. Centang Hutang Unit jika merupakan hutang unit.

  3. Setelah klik tombol Simpan maka data header akan tersimpan dan muncul Form Detail Kas Bank seperti di bawah ini :

    4. Lengkapi data detail kas bank kemudian klik tombol Simpan. Pilih Ya pada Hutang Unit jika merupakan hutang unit.

    Saat akan memasukkan no. invoice akan muncul form pencarian seperti di bawah ini :

  4. Setelah data detail kas bank disimpan maka transaksi ini akan muncul pada Tabel Kas Bank seperti di bawah ini :

  5. Klik icon untuk edit detail kas bank yang dimaksud.
  6. Klik icon untuk hapus detail transaksi kas bank.
  7. Untuk memastikan transaksi kas bank sudah balance maka klik icon maka akan muncul semua list transaksi kas bank yang ada.

    8. Atau lakukan pencarian dengan memasukkan no transaksi kas bank seperti di bawah ini :

    Berikut hasil pencariannya :

    Jika pada kolom Balance sudah menunjukkan angka 0 maka transaksi kas bank tersebut sudah balance.

    Jika masih belum balance maka pada kolom Balance akan muncul angka sebesar selisihnya tersebut.

  8. Klik icon untuk edit data header kas bank jika belum dilakukan proses posting.
  9. Klik icon untuk hapus data header kas bank jika belum dilakukan proses posting.
  10. Klik icon untuk posting transaksi kas bank, akan muncul konfirmasi seperti di bawah ini :

    11. Klik OK untuk posting transaksi kas bank. Tekan Cancel untuk membatalkan posting transaksi kas bank.

    Jika transaksi sudah posting maka icon akan berubah menjadi berwana hijau dan tidak dapat dilakukan edit data :

  11. Klik icon untuk melihat transaksi kas bank dalam format PDF seperti di bawah ini :

  12. Klik icon untuk melihat detail transaksi kas bank seperti di bawah ini :

Catatan:

  1. Jika Hutang Unit pada Header dicentang (langkah 2) dan pada Detail pilih Ya untuk Hutang Unit (langkah 3) maka akan terjadi jurnal otomatis.
  2. Jika Hutang Unit pada Header dicentang (langkah 2) dan pada Detail pilih Tidak untuk Hutang Unit (langkah 3) maka jurnal otomatisnya bernilai 0 (hanya ada untuk Headernya saja).
  3. Jika Hutang Unit pada Header tidak dicentang (langkah 2) dan pada Detail pilih Ya untuk Hutang Unit (langkah 3) maka tidak akan terbentuk jurnal otomatis.
