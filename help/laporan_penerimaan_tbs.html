LAPORAN PENERIMAAN TBS

Modul : Pabrik
Tentang : Laporan Penerimaan TBS
Isi :

Langkah-langkah meihat Laporan Penerimaan TBS :

Masuk ke menu, akan terdapat 3 form laporan yaitu Laporan Penerimaan TBS, Laporan Penerimaan TBS per Afdeling per Tanggal dan Laporan PenerimaanTBS per Afdeling per Bulan.

  1. Laporan Penerimaan TBS
    1. Pada form Laporan Penerimaan TBS pilih pabrik, tanggal, TBS dan Unit/Supplier kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Penerimaan TBS :

  2. Laporan Penerimaan TBS/Afdeling/Tanggal
    1. Pada form Laporan Penerimaan TBS/Afdeling/Tanggal pilih pabrik, tanggal, unit, dan afdeling kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Penerimaan TBS/Afdeling/Tanggal :

  3. Laporan Penerimaan TBS/Afdeling/Bulan
    1. Pada form Laporan Penerimaan TBS/Afdeling/Bulan pilih pabrik, unit dan periode kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Penerimaan TBS/Afdeling/Bulan :
