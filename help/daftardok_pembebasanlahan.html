DAFTAR DOKUMEN PEMBEBASAN LAHAN

Modul : PAD/GIS
Tentang : Daftar Dokumen Pembebasan Lahan
Isi :

Langkah-langkah melihat Daftar Dokumen Pembebasan Lahan :

  1. Masuk ke menu, pilih kebun kemudian klik tombol Preview untuk menampilkan daftar :

    2. Klik tombol Excel untuk menampilkan daftar dalam format Excel.

  2. Berikut tampilan dari Daftar Dokumen Pembebasan Lahan :

  3. Klik icon untuk menampilkan dokumen pembebasan lahan dalam format PDF.
