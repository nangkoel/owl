LAPORAN KEUANGAN BUDGET

Modul : Anggaran
Tentang : Laporan Keuangan Budget
Isi :

Langkah-langkah melihat Laporan Keuangan Budget :

Masuk ke menu, akan muncul 5 Tab yaitu Tab Neraca, Laba Rugi, Proyeksi Arus Kas Sebaran, Proyeksi Arus Kas per PT dan Tab Proyeksi Arus Kas seperti di bawah ini :

  1. Tab Neraca
    1. Pilih Tahun Budget, Perusahaan dan Unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Neraca :

  2. Tab Laba Rugi
    1. Klik Tab Laba Rugi, pilih Tahun Budget, Perusahaan dan Unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Laba Rugi :

  3. Tab Proyeksi Arus Kas Sebaran
    1. Klik Tab Proyeksi Arus Kas Sebaran, Pilih Tahun Budget, Perusahaan dan Unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Proyeksi Arus Kas Sebaran :

  4. Tab Proyeksi Arus Kas per PT
    1. Klik Tab Proyeksi Arus Kas per PT, pilih Tahun Budget dan Unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Proyeksi Arus Kas per PT :

  5. Tab Proyeksi Arus Kas
    1. Klik Tab Proyeksi Arus Kas, pilih Tahun Budget kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Proyeksi Arus Kas :
