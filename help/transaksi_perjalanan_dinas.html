Masuk ke menu, akan terdapat 2 tab yaitu Tab Form dan Tab List Data seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data perjalanan dinas.

    2. Akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses simpan.

  2. Tab List Data
    1. Klik Tab List Data, akan muncul list data perjalanan dinas. Terdapat juga fasilitas pencarian berdasarkan no. transaksi.

      2. Pengajuan perjalanan dinas ini masih belum disetujui oleh atasan dan pihak HRD. Atasan dan pihak HRD masih dapat diiganti.

    2. Klik icon untuk melihat Surat Perjalanan Dinas dalam format PDF. Statusnya masih belum disetujui oleh atasan danpihak HRD.
    3. Klik icon untuk hapus data perjalanan dinas.
    4. Klik icon untuk edit data perjalanan dinas.
