LAPORAN NILAI PEMBELIAN PER KELOMPOK MATERIAL

Modul : Gudang Laporan
Tentang : Laporan Nilai Pembelian Per Kelompok Material
Isi :

Langkah-langkah melihat Laporan Nilai Pembelian Per Kelompok Material :

  1. Masuk ke menu, pilih periode kemudian klik tombol Preview untuk menampilkan laporan.

    2. Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

  2. Berikut tampilan dari Laporan Nilai Pembelian Per Kelompok Material :

  3. Klik pada salah satu baris kelompok barang di atas untuk melihat detail laporan per kelompok barang seperti di bawah ini :

    4. Klik tombol Back untuk kembali ke halaman sebelumnya atau klik tombol Excel untuk melihat detail dalam format Excel.

  4. Klik pada nilai pembelian untuk melihat detail nominalnya :

    5. Klik tombol Back untuk kembali ke halaman sebelumnya atau klik tombol Excel untuk melihat detail dalam format Excel.
