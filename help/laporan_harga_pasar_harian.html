LAPORAN HARGA PASAR HARIAN

Modul : Pemasaran
Tentang : Laporan Harga Pasar Harian
Isi :

Langkah-langkah meihat Laporan Harga Pasar Harian :

Masuk ke menu, akan terdapat 2 tab yaitu tab Harga per Pasar dan tab Perbandiangan Harga seperti di bawah ini :

  1. Tab Harga per Pasar
    1. Klik Tab Harga per Pasar.

      2. Pilih pasar dan periode kemudian klik tombol Preview untuk menampilkan laporan.

      Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel.

    2. Berikut tampilan dari Laporan Harga per Pasar :

    3. Klik tombol Jpgraph untuk melihat grafik laporan berikut :

  2. Tab Perbandingan Harga
    1. Klik Tab Perbandingan Harga.

      2. Pilih pasar, komoditi dan periode kemudian klik tombol Preview untuk menampilkan laporan.

      Laporan ini dapat dibuka dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Perbandingan Harga :

    3. Klik tombol Jpgraph untuk melihat grafik laporan berikut :
