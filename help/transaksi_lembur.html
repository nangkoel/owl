TRANSAKSI LEMBUR

Modul : SDM
Tentang : Transaksi Lembur
Isi :

Langkah-langkah input Transaksi Lembur :

  1. Masuk ke menu, klik icon untuk membuat transaksi lembur baru. Berikut form Kop pengisian lembur :

    2. Pilih Kode Organisasi dan tanggal kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Setelah Kop disimpan maka akan muncul form rincian seperti di bawah ini :

    3. Lengkapi data seperti di atas kemudian klik icon untuk menyimpan rincian data. Jika sudah selesai mengisi rincian data, klik tombol Selesai.

  3. Data rincian yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

  4. Klik icon untuk edit rincian data lembur.
  5. Klik icon untuk hapus rincian data lembur.
  6. Klik icon untuk melihat semua list data lembur.
  7. Terdapat fasilitas pencarian berdasarkan nama perusahan dan/atau tanggal seperti di bawah ini :

    8. Klik tombol Cari untuk melakukan pencarian.

  8. Berikut hasil pencariannya :

  9. Klik icon untuk edit rincian data lembur (data Kop tidak dapat diedit).
  10. Klik icon untuk hapus data lembur.
  11. Klik icon untuk melihat data lembur dalam format PDF :
