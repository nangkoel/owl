TRANSAKSI ABSENSI

Modul : SDM
Tentang : Transaksi Absensi
Isi :

Langkah-langkah input Transaksi Absensi :

  1. Masuk ke menu, klik icon untuk membuat transaksi absensi baru.

    2. Berikut tampilan form Kop pengisian absensi :

    Kode organisasi dikunci berdasarkan subbagian user karyawan.

    1. Jika user karyawan kebun tidak memiliki subbagian, maka tampilannya sebagai berikut :

    2. Jika user karyawan memiliki subbagian, maka tampilannya sebagai berikut :

    Pilih Kode Organisasi, tanggal dan periode kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Setelah Kop disimpan maka akan muncul form rincian seperti di bawah ini :

    3. Lengkapi data seperti di atas kemudian klik icon untuk menyimpan rincian data. Jika sudah selesai mengisi rincian data, klik tombol Selesai.

  3. Data rincian yang sudah tersimpan akan muncul pada form Data Tersimpan di bawah ini :

  4. Klik icon untuk edit rincian data absensi.
  5. Klik icon untuk hapus rincian data absensi.
  6. Klik icon untuk melihat semua list data absensi.
  7. Terdapat fasilitas pencarian berdasarkan nama perusahan dan/atau tanggal seperti di bawah ini :

    8. Klik tombol Cari untuk melakukan pencarian.

  8. Berikut hasil pencariannya :

  9. Klik icon untuk edit rincian data absensi (data Kop tidak dapat diedit).
  10. Klik icon untuk hapus data absensi.
  11. Klik icon untuk melihat data absensi dalam format PDF :
