SETUP DAFTAR PERKIRAAN

Modul : Keuangan
Tentang : Setup Daftar Perkiraan
Isi :

Langkah-langkah Setup Daftar Perkiraan :

  1. Masuk ke menu, lengkapi form inputan seperti di bawah ini :

    2. Klik tombol Simpan untuk menyimpan data perkiraan di atas.

    Keterangan:

    1. Untuk pengisian Tingkatan :

    2. Untuk pemilihan Rincian dan Kas Bank :

      3. - Rincian : Untuk data perkiraan yang digunakan pada semua transaksi termasuk transaksi kas bank.
      - Kas Bank : Untuk data perkiraan yang digunakan pada transaksi kas bank saja.

    3. Untuk pengisian field aktif :

      4. Digit 1 = Kegiatan
      Digit 2 = Aset
      Digit 3 = Barang
      Digit 4 = Karyawan
      Digit 5 = Customer
      Digit 6 = Supplier
      Digit 7 = Kendaraan

    4. Untuk pengisian pemilik :

      5. Diisi dengan GLOBAL, HOLDING, KANWIL, KEBUN, PABRIK atau nama unit tertentu sebagai penanda kepemilikan data perkiraan tersebut.

  2. Akan muncul konfirmasi seperti di bawah ini:

    3. Klik tombol OK untuk menyimpan data perkiraan atau kllik tombol Cancel untuk mebatalkan penyimpanan.

  3. Semua data perkiraan akan muncul pada List Data seperti di bawah ini :

  4. Klik untuk edit data perkiraan.
  5. Klik untuk hapus data perkiraan.
  6. Klik untuk melihat list data perkiraan dalam format PDF.
  7. Klik untuk print list data perkiraan.
