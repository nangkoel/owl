Modul : Pengadaan
Tentang : Proses Tutup Buku Fisik

Langkah-langkah Proses Tutup Buku Fisik :

  1. Masuk ke menu, pilih Gudang kemudian klik tombol Simpan untuk menampilkan daftar proses yang berisi semua list barang yang ada di gudang. Jika sudah klik tombol Simpan dan ingin mengganti pilihan gudangnya maka klik tombol Ganti untuk mengganti pilihan gudang tersebut.

  2. Berikut tampilan dari Daftar Proses :

  3. Klik tombol Proses di atas maka akan muncul konfirmasi seperti di bawah ini :

    4. Klik tombol OK untuk melakukan proses tutup buku fisik atau klik tombol Cancel untuk membatalkannya.

  4. Setelah Klik tombol OK maka akan muncul konfirmasi lagi seperti di bawah ini :

    5. Klik tombol OK maka akan muncul pesan seperti di bawah ini :
