PURCHASE REQUEST APPROVAL

Module : Procurement
About : Purchase Request(PR) Approval Transaction
Isi :

Steps to approve a purchase request :

  1. Get in to the menu, it will display all purchase request that require approval from the relevant Manager.

  2. At this stage, the purchase request can be approved, rejected all PR or rejected just a few items.
  1. The steps to approve the purchase request :

    2. On list of PR, click the Approve, the following is a the display approval form:

    Purchase request can be submitted for the next verification by submitting to the next parties with notes.

    If approval only get one approval, fill the note and then directly click the button to Purchasing

    Then the purchase request will be processed by Purcasing Dept.

  2. The steps to reject a purchase request :

    3. On the list of purchase requests, click Reject then the following form will appear:

    Fill a note to reject the purchase request.

  3. The steps to reject some purchase request :

    4. Click Reject some and will display the following form :

    Reject some typically done for PR that has more than one item of goods.

    Fill the reason of rejected, then click Reject, and then click Done to immediately processed by the Purchasing.
