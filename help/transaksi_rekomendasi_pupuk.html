TRANSAKSI REKOMENDASI PUPUK

Modul : Kebun
Tentang : Transaksi Rekomendasi Pupuk
Isi :

Langkah-langkah Input Transaksi Rekomendasi Pupuk :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Tahun tanam akan muncul secara otomatis setelah memilih blok. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkannya.

  2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  3. Klik icon untuk melihat list data dalam format PDF.
  4. Klik icon untuk melihat list data dalam format Excel.
  5. List Data tersebut juga dapat dilihat dengan klik icon
  6. Dapat dilakukan pencarian berdasarkan kebun dan tahun pupuk seperti di bawah ini :

    7. Pilih Kebun dan/atau Tahun Pupuk kemudian klik tombol Cari.

  7. Berikut hasil pencariannya :

  8. Klik icon untuk edit data rekomendasi pupuk.
  9. Klik icon untuk hapus data rekomendasi pupuk.
