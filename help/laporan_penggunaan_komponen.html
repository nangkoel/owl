LAPORAN PENGGUNAAN KOMPONEN

Modul : Traksi
Tentang : Laporan Penggunaan Komponen
Isi :

Langkah-langkah melihat Laporan Penggunaan Komponen :

  1. Masuk ke menu, pilih unit, jenis vehicle, kode kendaraan, tanggal dari dan tanggal sampai kemudian klik tombol Simpan untuk menampilkan laporan.

    2. Jika sudah klik tombol Simpan dan ingin mengganti pilihan data, klik tombol Ganti untuk mengganti pilihan.

  2. Berikut tampilan dari Laporan Penggunaan Komponen :

  3. Laporan ini dapat dibuka dalam format Excel dan PDF dengan klik icon atau klik icon di atas.
