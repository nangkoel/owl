MASTER DATA MESIN, KENDARAAN DAN ALAT BERAT

Modul : Traksi
Tentang : Input master mesin, kendaraan dan alat berat
Isi :

Langkah-langkah input master mesin,kendaraan, dan alat berat :

  1. Masuk ke menu, pilih kode kelompok Kendaraan, Mesin atau Alat Berat.
  2. Input kode organisasi pemilik kendaraan, mesin atau alat berat tersebut.
  3. Pilih nama barang yang sesuai dengan alat, kendaraan atau mesin.
  4. Input Nomor akun, nomor rangka, tambah detail jika ada detail tambahan.
  5. Input tipe mesin dan kendaraan sesuai dengan data yang ingin di inputkan.
  6. Kode alat (nomor polisi), disarankan penginputan nomor polisi atau kode alat, kendaraan yang informatif
    Contoh :
    Untuk no polisi kendaraan disambung semua BG66699FM.
    Untuk mesin genset, GEN01-SMLE.
    Hindari penggunaan spasi.
  7. Input tahun perolehan, tahun berapa di dapatkan kendaraan, mesin atau alat berat tersebut.
  8. Inputkan data berat kosong dan nomor mesin jika ada dan tertera.
  9. Kepemilikan, masukan milik sendiri atau sewa.
  10. Kode traksi pilih Traksi pemilik kendaraan, mesin atau alat berat tersebut.
