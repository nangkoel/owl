TRANSAKSI TIMBANGAN PEMBELI

Modul : Pabrik
Tentang : Transaksi Timbangan Pembeli
Isi :

Langkah-langkah input Transaksi Timbangan Pembeli :

  1. Masuk ke menu, akan muncul form seperti di bawah ini :

    2. Pilih nama barang, nama pelanggan, dan no.kontrak kemudian klik tombol Preview untuk memunculkan list data.

  2. Berikut tampilan dari list data :

    3. Isikan berat bersin pelanggan kemudian klik tombol Simpan untuk menyimpan per transaksi atau klik tombol Simpan Seluruhnya untuk menyimpan seluruh transaksi sekaligus setelah mengisikan semua berat bersih pelanggannya.

  3. Setelah klik tombol Simpan maka akan muncul konfirmasi di bawah ini :

    4. Klik tombol OK untuk menyimpan.

  4. Setelah data disimpan maka baris tersebut akan berwarna hijau seperti di bawah ini :

  5. Terdapat fasilitas pencarian timbangan pembeli berdasarkan pada no. transaksi dan/atau no. kontrak dan/atau tanggal seperti di bawah ini :

    6. Isikan data yang ingin dicari kemudian klik tombol Cari untuk melakukan proses pencarian.

  6. Berikut hasil pencariannya :

  7. Klik icon untuk edit data timbangan pembeli.
  8. Klik icon untuk menggunakan timbangan loco (timbangan penjual/berat bersih PKS), akan muncul konfirmasi seperti di bawah ini :

    9. Klik OK untuk menggunakan timbangan loco, maka berat bersih pelanggan = berat bersih PKS(KG) seperti di bawah ini :
