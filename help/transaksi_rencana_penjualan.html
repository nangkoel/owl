TRANSAKSI RENCANA PENJUALAN

Modul : Pemasaran
Tentang : Transaksi Rencana Penjualan
Isi :

Langkah-langkah Transaksi Rencana Penjualan :

Masuk ke menu, akan terdapat 2 tab yaitu tab Kop dan tab Detail Rencana Jual seperti di bawah ini :

  1. Tab Kop
    1. Klik tab Kop, pilih periode dan kode organisasi kemudian klik tombol Simpan untuk menyimpan data Kop rencana penjualan.

    2. Saat data disimpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK maka tampilan Kop akan menjadi seperti di bawah ini :

      Klik tombol Selesai jika sudah selesai input Kop rencana penjualan.

    3. Data Kop rencana penjualan yang sudah tersimpan akan muncul pada form Data Tersimpan seperti di bawah ini :

    4. Klik icon untuk edit Kop rencana penjualan.
    5. Klik icon untuk hapus Kop rencana penjualan.
    6. Klik icon untuk melihat data Kop dalam format PDF.
  2. Tab Detail Rencana Jual
    1. Klik tab Detail Rencana Jual.

      2. Isi tanggal, nama barang, kode pelanggan, lokasi pengiriman dan volume kemudian klik tombol Simpan untuk menyimpan detail atau klik tombol Batal untuk membatalkan.

    2. Data detail yang tersimpan akan muncul pada form Data Tersimpan seperti di bawah ini :

    3. Klik icon untuk edit detail rencana penjualan.
    4. Klik icon untuk hapus detail rencana penjualan.
