SETUP PELANGGAN

Modul : Pemasaran
Tentang : Setup Pelanggan
Isi :

Langkah-langkah Setup Pelanggan :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Untuk pengisian Nama Grup Pelanggan, klik icon maka akan muncul form seperti di bawah ini :

    3. Ketikkan nama grup pelanggan yang dicari kemudian klik tombol Find. Klik pada nama grup pelanggan yang dimaksud.

  3. Untuk pengisian Nama Akun, klik icon maka akan muncul form seperti di bawah ini :

    4. Ketikkan nama akun yang dicari kemudian klik tombol Find. Klik pada nama akun yang dimaksud.

  4. Data yang sudah tersimpan akan muncul pada daftar di bawah ini :

  5. Klik icon untuk edit data pelanggan.
  6. Klik icon untuk hapus data pelanggan.
