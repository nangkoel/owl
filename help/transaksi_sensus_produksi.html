TRANSAKSI SENSUS PRODUKSI

Modul : Kebun
Tentang : Transaksi Sensus Produksi
Isi :

Langkah-langkah Transaksi Sensus Produksi :

  1. Masuk ke menu, lengkapi data pada form Kontrol seperti di bawah ini :

    2. Klik tombol Rincian untuk menampilkan form Detail.

  2. Berikut tampilan form detail :

    3. Lengkapi data pada form detail seperti di atas kemudian klik tombol Simpan untuk simpan detail atau klik tombol Batal untuk membatalkan.

  3. Data detail yang sudah disimpan akan muncul pada Daftar Rencana Panen di bawah ini :

  4. Klik icon untuk edit data detail sensus produksi.
  5. Klik icon untuk hapus data detail sensus produksi.
  6. Klik icon untuk meihat semua list data sensus produksi atau lakukan pencarian data seperti di bawah ini :

    7. Klik tombol Cari untuk melakukan pencarian.

  7. Berikut hasil dari pencarian di atas :

  8. Klik icon pada form cetak untuk mencetak list data transaksi sensus produksi.
  9. Klik icon pada form cetak untuk melihat list transaksi sensus produksi dalam format PDF.
  10. Klik icon untuk edit data transaksi sensus produksi.
  11. Klik icon untuk menghapus data transaksi sensus produksi.
