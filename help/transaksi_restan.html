TRANSAKSI RESTAN

Modul : Kebun
Tentang : Transaksi Restan
Isi :

Langkah-langkah Transaksi Restan :

Masuk ke menu, akan terdapat 2 tab yaitu tab input restan dan tab laporan restan seperti di bawah ini :

  1. Tab Input Restan
    1. Klik Tab Input Restan

      2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data restan atau klik tombol Batal untuk membatalkan.

    2. Data yang sudah disimpan akan muncul pada Form Data Tersimpan di bawah ini :

    3. Untuk menampilkan data yang dimaksud dapat memilih periode dan blok pada form Data Tersimpan seperti di bawah in :

    4. Klik icon untuk edit data restan.
    5. Klik icon untuk hapus data restan.
  2. Tab Laporan Restan
    1. Klik Tab Laporan Restan

      2. Lengkapi data seperti di atas kemudian klik tombol Preview untuk melihat Laporan Restan.

    2. Berikut tampilan dari Laporan Restan :

    3. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.
