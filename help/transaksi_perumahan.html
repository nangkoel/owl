TRANSAKSI PERUMAHAN

Modul : Umum
Tentang : Transaksi Perumahan
Isi :

Langkah-langkah input Transaksi Perumahan :

Masuk ke menu, akan muncul 3 Tab yaitu Tab Rumah, Tab Prabot dan Tab Penghuni seperti di bawah ini :

  1. Tab Rumah
    1. Lengkapi data seperti di bawah ini :

    2. Saat klik tombol Simpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK.

    3. Data yang sudah tersimpan akan muncul pada Form Data Tersimpan di bawah ini :

    4. Klik icon untuk edit data perumahan.
    5. Klik icon untuk hapus data perumahan.
  2. Tab Prabot
    1. Klik Tab Prabot, lengkapi data seperti di bawah ini :

    2. Saat klik tombol Simpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK.

    3. Data yang sudah tersimpan akan muncul pada Form Data Tersimpan di bawah ini :

    4. Klik icon untuk edit data prabot.
    5. Klik icon untuk hapus data prabot.
  3. Tab Penghuni
    1. Klik Tab Penghuni, legkapi data seperti di bawah ini :

    2. Saat klik tombol Simpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK.

    3. Data yang sudah tersimpan akan muncul pada Form Data Tersimpan di bawah ini :

    4. Klik icon untuk edit data penghuni.
    5. Klik icon untuk hapus data penghuni.
