LAPORAN PRODUKSI PKS

Modul : Anggaran
Tentang : Laporan Produksi PKS
Isi :

Langkah-langkah melihat Laporan Produksi PKS :

  1. Masuk ke menu, pilih tahun budget dan nama pabrik kemudian klik tombol Proses untuk menampilkan laporan.

  2. Berikut tampilan dari Laporan Produksi PKS :

  3. Klik icon untuk melihat laporan dalam format Excel.
  4. Klik icon untuk melihat laporan dalam format PDF.
