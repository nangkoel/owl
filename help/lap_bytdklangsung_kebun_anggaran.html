Modul : Anggaran
Tentang : Laporan Biaya Tidak Langsung Kebun
Isi :

Langkah-langkah melihat Laporan Biaya Tidak Langsung Kebun :

  1. Masuk ke menu, pilih anggaran tahunan dan kode organisasi kemudian klik tombol Simpan untuk menampilkan laporan.

  2. Berikut tampilan dari laporan biaya tidak langsung kebun :

  3. Klik icon untuk melihat laporan dalam format Excel.
  4. Klik icon untuk melihat laporan dalam format PDF.
