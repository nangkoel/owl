LAPORAN PRODUKSI

Modul : Kebun
Tentang : Laporan Produksi
Isi :

Langkah-langkah Laporan Produksi :

Masuk ke menu, akan terdapat 3 Tab seperti di bawah ini :

  1. Tab Laporan Produksi Kebun
    1. Pilih periode, TBS (External, Internal, Afiliasi), dan unit/supplier kemudian klik tombol Preview.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Produksi Kebun :

  2. Tab Laporan Produksi Kebun Detail
    1. Klik tab Laporan Produksi Kebun Detail. Pilih periode dan unit kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Produksi Kebun Detail :

  3. Tab Trend Produksi
    1. Pilih Periode dan Unit kemudian klik tombol Preview untuk menampilkan Laporan dalam format HTML atau klik tombol Excel untuk menampilkan Laporan dalam format Excel.

    2. Berikut tampilan dari Trend Produksi :
