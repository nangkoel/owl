ACTIVE/ DEACTIVE/DELETE USER

Modul : Administrator
Tentang : Active/Deactive/Delete User
Isi :

Langkah-langkah Active/Deactive/Delete User :

  1. Masuk ke menu, ketikkan user yang akan diaktifkan atau dinonaktifkan atau dihapus kemudian klik tombol Find :

  2. Tick check box pada kolom Status Active/NotActive untuk mengaktifkan user.
  3. Untick check box pada kolom Status Active/NotActive untuk menonaktifkan user.
  4. Klik icon untuk menghapus user.
