TRANSAKSI FAKTUR PAJAK

Modul : Pemasaran
Tentang : Transaksi Faktur Pajak
Isi :

Langkah-langkah Transaksi Faktur Pajak :

Masuk ke menu, akan terdapat 2 tab yaitu tab Buat Faktur dan tab Daftar Faktur seperti di bawah ini :

  1. Tab Buat Faktur
    1. Klik Tab Buat Faktur kemudian lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan faktur atau klik tombol Batal untuk membatalkan.

  2. Tab Daftar Faktur
    1. Klik Daftar Faktur yang berisi semua faktur yang sudah dibuat.
    2. Terdapat fasilitas pencarian faktur berdasarkan no. kontrak, bulan dan rekanan seperti di bawah ini :

    3. Klik icon untuk melakukan posting faktur pajak, akan muncul konfirmasi seperti di bawah ini :

      4. Klik tombol OK. Saat posting berhasil maka akan muncul pemberitahuan berikut :

      Klik tombol OK, maka tampilan daftar fakturnya menjadi seperti di bawah ini :

    4. Klik icon untuk hapus faktur pajak.
    5. Klik icon untuk melihat faktur pajak dalam format PDF :
