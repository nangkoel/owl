POSTING TRANSAKSI PEMAKAIAN BARANG

Modul : Pengadaan
Tentang : Posting Transaksi Pemakaian Barang
Isi :

Langkah-langkah posting transaksi pemakaian barang:

1. Masuk ke menu kemudian isikan Gudang dan klik tombol Simpan maka akan muncul tampilan seperti dibawah ini:

2. Lakukan pencarian no. transaksi pada tab "Blm Posting". Berikut tampilannya:

3. Klik tombol Proses untuk melakukan proses posting transaksi tersebut. Akan muncul window berikut:

Klik tombol Posting, akan muncul konfirmasi seperti di bawah ini:

Klik OK maka akan muncul pesan berikut:

4. Klik Done maka transaksi sudah terposting.
