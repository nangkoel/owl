LAPORAN PERAWATAN MESIN

Modul : Pabrik
Tentang : Laporan Perawatan Mesin
Isi :

Langkah-langkah meihat Laporan Perawatan Mesin :

  1. Masuk ke menu, pilih pabrik, station dan periode kemudian klik tombol Simpan untuk menampilkan laporan.

    2. Jika sudah menekan tombol Simpan tetapi ingin mengubah pilihan maka klik tombol Ganti untuk mengganti pilihan tersebut.

  2. Berikut tampilan dari Laporan Perawatan Mesin :

  3. Laporan ini dapat dibuka dalam format Excel dan format PDF dengan klik icon
