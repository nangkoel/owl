TRANSAKSI DAFTAR ASSET

Modul : Keuangan
Tentang : Transaksi Daftar Asset
Isi :

Langkah-langkah Transaksi Daftar Asset :

Masuk ke menu maka akan terdapat 2 tab yaitu tab Form Input Asset dan tab List Data.

  1. Tab Form Input Asset
    1. Klik tab Form Input Asset atau klik icon untuk input data asset. Berikut tampilan form nya :

      2. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk simpan data atau klik tombol Batal untuk membatalkan input data asset.

    2. Setelah klik simpan akan muncul konfirmasi seperti di bawah ini :

      3. Klik OK maka akan muncul pesan seperti di bawah ini :

      Data asset sudah tersimpan.

  2. Tab List Data
    1. Klik tab List Data atau klik icon untuk melihat semua list data asset.

    2. Dapat dilakukan pencarian untuk list data asset dengan pencarian berdasarkan no.asset atau berdasarkan nama asset. Berikut tampilannya :

      3. Klik tombol cari untuk melakukan pencarian.

    3. Berikut hasil pencariannya :

    4. Klik icon untuk edit data asset.
    5. Klik icon untuk hapus data asset.
