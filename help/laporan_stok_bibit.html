Modul : Kebun
Tentang : Laporan Stok Bibit
Isi :

Langkah-langkah Laporan Stok Bibit :

Masuk ke menu, akan muncul 2 tab seperti di bawah ini :

  1. Tab Laporan Stok Bibit
    1. Pilih Unit dan Batch kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini juga dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau Excel di atas.

    2. Berikut tampilan dari Laporan Stok Bibit :

  2. Tab Rekap Stok Bibit
    1. Pilih Perusahaan, Kebun dan s/d Tanggal kemudian klik tombol Preview untuk menampilkan Rekap dalam format HTML atau klik tombol Excel untuk menampilkan Rekap dalam format Excel.
