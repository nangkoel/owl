PENGEMBALIAN BARANG KE GUDANG (RETUR)

Modul : Pengadaan
Tentang : Pengembalian Barang ke Gudang (Retur)
Isi :

Langkah-langkah pengembalian barang ke gudang (retur) :

Masuk ke menu, akan terdapat form daftar gudang dan 2 tab yaitu tab Pengembalian/retur dan List Data.

  1. Form Daftar Gudang

    2. Pilih gudang yang dimaksud kemudian klik Simpan. Tombol Ganti digunakan untuk mengganti pilihan gudang.

  2. Tab Pengembalian/retur
    1. Setelah klik tombol Simpan di atas maka secara otomatis No. Dokumen pengembalian/retur barang akan terbentuk kemudian isikan Tanggal Retur, No. Dokumen Lama, Kode Barang, dan Kode Blok seperti tampilan di bawah ini :

    2. Klik tombol Check di atas untuk cek no. dokumen lama, jika no. dokumen tersebut ada pada database maka nama barang, jumlah dan satuannya akan ditampilkan seperti di bawah ini :

    3. Kemudian isikan kuantitas barang yang akan dikembalikan beserta keterangannya kemudian klik tombol Simpan.

  3. Tab List Data
    1. Klik Tab List Data maka akan muncul semua pengembalian/retur barang di gudang. Lakukan pencarian untuk transaksi pengembalian/retur barang yang dimaksud seperti tampilan di bawah ini :

    2. Berikut hasil pencarian transaksi di atas :

    3. Klik icon untuk hapus transaksi pengembalian/retur barang. Akan muncul konfirmasi hapus seperti dibawah ini:

      4. Klik OK untuk hapus transaksi pengembalian/retur.

    4. Klik icon untuk melihat Dokumen Retur Barang dalam format PDF.

      5. Karena transaksi pengembalian/retur barang ini masih belum diposting maka status Dokumen Retur Barang ini masih "Not Confirm".
