TRANSAKSI PROMOSI/DEMOSI/MUTASI

Modul : SDM
Tentang : Transaksi Promosi/Demosi/Mutasi
Isi :

Langkah-langkah input Transaksi Promosi/Demosi/Mutasi :

Masuk ke menu, akan terdapat 2 tab yaitu Tab Form dan Tab List Data seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form kemudian lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data.

  2. Tab List Data
    1. Klik List Data. Klik tombol Cari maka semua data promosi/demosi/mutasi yang sudah disimpan akan muncul seperti di bawah ini :

    2. Terdapat fasilitas pencarian berdasarkan no. transaksi kemudian klik tombol Cari untuk melakukan pencarian seperti di bawah ini :

    3. Klik icon untuk melihat surat keputusan promosi/demosi/mutasi dalam format PDF.
    4. Klik icon untuk edit data promosi/demosi/mutasi.
    5. Klik icon untuk hapus data promosi/demosi/mutasi.
