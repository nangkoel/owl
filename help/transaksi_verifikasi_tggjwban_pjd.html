TRANSAKSI VERIFIKASI PERTANGGUNGJAWABAN PERJALANAN DINAS

Modul : SDM
Tentang : Transaksi Verifikasi Pertanggungjawaban Perjalanan Dinas
Isi :

Langkah-langkah input Transaksi Verifikasi Pertanggungjawaban Perjalanan Dinas :

Masuk ke menu, akan muncul 2 Tab yaitu Tab List Data dan Tab Form seperti di bawah ini :

  1. Tab List Data
    1. Klik Tab List Data, akan muncul semua list data pertanggungjawaban perjalanan dinas yang harus diverifikasi.
    2. Terdapat fasilitas pencarian berdasarkan no. transaksi seperti di bawah ini :

    3. Klik icon untuk melakukan verifikasi pertanggungjawaban perjalanan dinas pada Tab Form
  2. Tab Form
    1. Berikut tampilan dari form verifikasi :

    2. Klik icon untuk menampilkan nominal biaya pertanggungjawaban perjalanan dinas pada kolom disetujui.
    3. Setelah nominal muncul pada kolom disetujui, klik icon untuk menyimpan verifikasi tersebut.
    4. Jika sudah selesai melakukan verifikasi, klik tombol Selesai.

    5. Pertanggungjawaban perjalanan dinas sudah disetujui, dapat dicek pada Tab List Data :

    6. Klik icon untuk melihat biaya yang sudah disetujui pada Surat Perjalanan Dinas :
